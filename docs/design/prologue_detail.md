# 《末世行商》序章详细设计文档
### "第一道车辙" —— 教学流程 × 叙事设计

> **定位**：本文档基于已实现的教学架构，为 8 个叙事窗口填充具体的故事内容。
> 教学流程（接单→装货→驾驶→交货→篝火→探索→到达）保持不变，
> 本文档只重新设计**每个窗口里讲什么故事、传达什么情感**。

---

## 一、序章总览

| 项目 | 内容 |
|------|------|
| **标题** | 第一道车辙 |
| **游玩时长** | 约 10-15 分钟（与新手教学完全重合） |
| **核心目标** | ① 教会玩家跑商循环 ② 让玩家在意两位主角 ③ 种下一个悬念暗线 |
| **情感基调** | 平淡日常 → 共患难 → 微暖 → 好奇 |
| **设置 flag** | `sd_prologue_01_done`、`tutorial_started`，另含教学 flag 若干 |

### 序章的三层任务

```
┌─────────────────────────────────────┐
│  层 1: 系统教学                      │  ← 教学架构承担（不变）
│  (接单、跑商、篝火、探索)             │
├─────────────────────────────────────┤
│  层 2: 角色与情感                    │  ← 本文档的核心
│  (林砾的沉默关心、陶夏的明亮活力)     │
│  (两人从磨合到信任的微弧线)           │
├─────────────────────────────────────┤
│  层 3: 悬念播种                      │  ← 暗线，不抢戏
│  (收音机信号 → 西南方向 → 云海高塔)   │
└─────────────────────────────────────┘
```

### 叙事前提

**两人刚搭伙不久，一直在温室附近几个固定点跑短途送货。**

- 收入微薄，"油钱都快赚不回来了"
- 彼此还在磨合——陶夏话多，林砾沉默；分工已有默契，但还称不上亲密
- 麻薯号（货车）是她们的全部家当
- 她们从未走出温室社区周边的安全区域

**这次序章是她们第一次踏上更远的旅途。**

- 第一段路（greenhouse_farm → greenhouse）是熟路——日常
- 第二段路（greenhouse → tower）是第一次——冒险的开始
- 教学结束时，玩家和角色一起完成了"从舒适圈到未知世界"的跨越

## 四、叙事设计说明

### 4.1 情感弧线

```
①平淡日常 → ②轻松出发 → ③突遇危险 → ④安全感恢复
                                            ↓
               ⑧好奇+成就 ← ⑦新世界 ← ⑥决心出发 ← ⑤情感高点
```

序章的情感走向是一个**微型成长弧**：
- 前半段（①-④）：从舒适区出发 → 遭遇意外 → 回到安全的温室
- 转折点（⑤）：篝火夜谈，两人第一次聊"我们想走多远"
- 后半段（⑥-⑧）：带着新的勇气走出去 → 到达从未到过的地方

### 4.2 暗线（信号）的埋设节奏

暗线的原则是**不抢情感的戏**，只做背景里的异响，让玩家自然积累好奇心：

| 窗口 | 暗线动作 | 强度 |
|------|---------|------|
| ② 出发 | 林砾拨了一下收音机旋钮 | ○○○ 极轻，可能被忽略 |
| ④ 温室 | 沈禾提到"塔台在说信号干扰" | ○○○ 背景信息，不展开 |
| ⑤ 篝火 | 无——这场戏纯属于情感 | — |
| ⑥ 接单 | 林砾说"想去看看天线"，陶夏注意到"顺便" | ●○○ 开始留意 |
| ⑦ 驾驶 | 收音机响了，林砾手指停在旋钮上 | ●●○ 第三次，形成模式 |
| ⑧ 塔台 | 韩策正式提到"西南方周期性信号" + 林砾记录 | ●●● 暗线移交给主线 |

到序章结束，信号的存在已经被自然地建立了，但**没有回答任何问题**。
回答留给后续章节：
- Ch.1：韩策口中再次提到 → 确认不是噪音
- Ch.2：鸣砂广播说出"云海高塔" → 第一次命名
- Ch.2 末：林砾坦白"我一直在追这个信号" → 正式揭示个人动机

### 4.3 与主线各章的衔接

| 序章埋点 | 主线接续 |
|---------|---------|
| 韩策提到铁锁商队（⑧"比铁锁的人快"） | Ch.1 韩策详述铁锁 → Ch.3 正面遭遇 |
| 路上的伏击（③） | Ch.1 沈禾提到"路标被挪" → Ch.4-5 静潮社浮现 |
| 沈禾的随意但温暖的接待（④） | Ch.1-2 聚落关系深化，沈禾开始托付私事 |
| "看油够不够"（⑤） | 终章旅途回忆闪回 |
| 林砾记录本上的数字（⑧） | Ch.2 陶夏发现频率数字 → 林砾坦白 |
| 韩策要求记录信号时间位置（⑧） | Ch.2 四聚落态度收集 / Ch.6 地图拼合 |

### 4.4 序章结束时玩家的三层感受

```
表层（明确的）：  我学会跑商了——接单、送货、篝火、探路
中层（感受到的）：我喜欢这两个人。陶夏有趣，林砾温柔。想看她们继续走
底层（好奇的）：  那个信号是什么？林砾在记录什么？西南方有什么？
```

## 五、补充说明

### 5.1 收音机功能

收音机作为 UI 功能**保持原样**——玩家自行发现时有气泡教学。

序章中的信号不通过"玩家操作收音机"来传递，而是在**对话和驾驶事件文字里自然出现**：
- ② 出发：林砾拨旋钮（对话文字）
- ⑦ 驾驶：收音机自己响了（路上事件文字）

玩家不需要主动打开收音机就能感知到暗线。如果玩家后来自己打开收音机，会发现更多内容——这是奖励好奇心，不是惩罚忽略。

### 5.2 战斗设计

战斗系统是轻决策——玩家只有攻击和逃跑两个选项。

序章伏击（③）的叙事设计**不依赖战斗结果**：
- 打赢了：继续前行
- 跑了：也继续前行
- 无论哪种，事后都触发陶夏的那句"这条路不是一直都挺安全的吗？"

伏击本身的叙事价值不在于胜负，而在于**"熟悉的路变得不安全了"**——世界在变化。

## 附录 B：新手保护与反挫败设计

- 前 15 分钟教学期内，任何失败都不应导致"资源归零 + 长时间卡死"
- 初期坏结果以"少赚、绕路、轻微受损"为主，不出现严重断档
- 关键教学功能必须用固定事件触发，不完全依赖随机
- 如果玩家连续两趟亏损，自动提高下一趟基础订单利润或触发补偿事件

# 美术素材制作规范

> 统一画风：少女终末旅行（简笔画风格，柔和线条，白色背景）

---

## 一、通用画风基底

所有 prompt 共享前缀：

```
少女终末旅行画风，末世废土风格立绘，
```

共享后缀：

```
简笔画风格，柔和线条，白色背景
```

---

## 二、素材类型约束

### 2.1 CG / 背景图

基础约束：

```
少女终末旅行画风，末世废土风格立绘，
图层混合模式，手绘风格，大块色块，柔焦效果，色彩叠加，景深
简笔画风格，柔和线条，白色背景
```

**战斗背景图**额外约束：无任何人物角色，仅作为背景氛围。

**CG / 普通背景图**与特定人物关联时：
- 有角色人设参考图 → 必须参考人设参考图
- 无人设参考图 → 用任意角色人设参考图作为画风参考
- 需要用人设相关的 prompt 稳定人设（见下方角色 prompt）

### 2.2 角色立绘

- 必须生成**全身角色图**，无任何背景，不需要额外特效装饰
- 有角色人设参考图 → 必须参考

### 2.3 角色头像

- 有角色人设参考图 → 必须参考

### 2.4 敌人立绘

- 必须生成**全身角色图**，无任何背景，特效装饰可以有但不能太多
- 必须直接在 prompt 中输入敌人名称
- 有敌人人设参考图 → 必须参考

### 新增姿势差分模板

```lua
{
    prompt = "少女终末旅行画风，末世废土风格立绘，Q版二头身角色，[角色外貌描述]，[姿势描述]，[表情]，简笔画风格，柔和线条，白色背景",
    name = "chibi_[角色id]_[姿势]",
    target_size = "512x512",
    aspect_ratio = "1:1",
    transparent = true,
    reference_images = { "assets/image/chibi_[角色id]_[默认立绘].png" },
}
```

### Chibi Prompt 编写要点

- 必须包含"Q版二头身角色"确保比例一致
- 姿势描述用动作短语（"半蹲举枪"、"坐姿手持饭碗"），不要写复杂场景
- 必须引用角色默认 chibi 作为 reference_image，否则画风容易偏移
- 表情用一个四字词概括
- 战斗姿势的角色应面朝右侧（游戏中通过 scaleX 翻转控制朝向）

## 五、新角色 Prompt 模板

```
少女终末旅行画风，末世废土风格立绘，[性别年龄]，[名字]，[身份/职业]，[外貌特征1]，[外貌特征2]，[服装描述]，[表情/气质]，简笔画风格，柔和线条，白色背景
```

生成参数：

```lua
{
    prompt = "...",
    name = "portrait_xxx",
    target_size = "512x1024",
    aspect_ratio = "2:3",
    transparent = true,
}
```

### Prompt 编写要点

- 保持"少女终末旅行画风"前缀确保风格一致
- 外貌特征控制在 2-3 个，太多会互相干扰
- 服装是区分角色最有效的手段，优先描述
- 表情/气质用一个四字词概括即可
- 不写"末世废土"以外的背景描述，统一用"白色背景"配合透明底

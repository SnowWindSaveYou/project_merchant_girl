# 美术素材制作规范

> 统一画风：少女终末旅行（简笔画风格，柔和线条，白色背景）

---

## 一、通用画风基底

所有 prompt 共享前缀：

```
少女终末旅行画风，末世废土风格立绘，
```

共享后缀：

```
简笔画风格，柔和线条，白色背景
```

---

## 二、素材类型约束

### 2.1 CG / 背景图

基础约束：

```
少女终末旅行画风，末世废土风格立绘，
图层混合模式，手绘风格，大块色块，柔焦效果，色彩叠加，景深
简笔画风格，柔和线条，白色背景
```

**战斗背景图**额外约束：无任何人物角色，仅作为背景氛围。

**CG / 普通背景图**与特定人物关联时：
- 有角色人设参考图 → 必须参考人设参考图
- 无人设参考图 → 用任意角色人设参考图作为画风参考
- 需要用人设相关的 prompt 稳定人设（见下方角色 prompt）

### 2.2 角色立绘

- 必须生成**全身角色图**，无任何背景，不需要额外特效装饰
- 有角色人设参考图 → 必须参考

### 2.3 角色头像

- 有角色人设参考图 → 必须参考

### 2.4 敌人立绘

- 必须生成**全身角色图**，无任何背景，特效装饰可以有但不能太多
- 必须直接在 prompt 中输入敌人名称
- 有敌人人设参考图 → 必须参考

## 四、纸娃娃 (Chibi) Prompt

> 统一参数：比例 1:1，透明背景
> 必须用对应角色的默认 chibi 立绘作为参考图（reference_images），保持画风一致
>
> **尺寸标准**：
> - 主角姿势差分：512×512（已完成，保持不变）
> - NPC / 敌人 / 新增纸娃娃：**128×128**（约 15-25KB，节省包体积）
> - 已完成的素材不再重新生成，新素材统一按 128×128

### 通用 Prompt 前缀

```
少女终末旅行画风，末世废土风格立绘，Q版二头身角色，
```

### 新增姿势差分模板

```lua
{
    prompt = "少女终末旅行画风，末世废土风格立绘，Q版二头身角色，[角色外貌描述]，[姿势描述]，[表情]，简笔画风格，柔和线条，白色背景",
    name = "chibi_[角色id]_[姿势]",
    target_size = "128x128",              -- 新素材统一 128×128（主角差分已完成的保持 512）
    aspect_ratio = "1:1",
    transparent = true,
    reference_images = { "assets/image/chibi_[角色id]_[默认立绘].png" },
}
```

### Chibi Prompt 编写要点

- 必须包含"Q版二头身角色"确保比例一致
- 姿势描述用动作短语（"半蹲举枪"、"坐姿手持饭碗"），不要写复杂场景
- 必须引用角色默认 chibi 作为 reference_image，否则画风容易偏移
- 表情用一个四字词概括
- 战斗姿势的角色应面朝右侧（游戏中通过 scaleX 翻转控制朝向）

## 五、场景图制作规范

### 5.1 聚落停泊场景图（scene_*）

> 用于卡车停泊在聚落时的纸娃娃窗口背景，替代视差滚动层
> 统一参数：比例 16:9，target_size = "1376x768"，**不透明**

**构图规则**：
- **下方 1/3**：路面/地面（水泥、泥地、草地等，角色和卡车站在这个区域）
- **上方 2/3**：周边场景建筑、植被、天空等环境元素
- 整体透视：略俯视的远景或中景，给人"到达某个地方"的感觉
- 不画任何人物角色，仅作为纯背景

**Prompt 模板**：
```
少女终末旅行画风，末世废土风格立绘，[聚落名称/类型描述]，
图层混合模式，手绘风格，大块色块，柔焦效果，色彩叠加，景深，
下方三分之一为路面地面，上方为建筑和场景，无任何人物，
简笔画风格，柔和线条，白色背景
```

**生成参数**：
```lua
{
    prompt = "...",
    name = "scene_[聚落id]",
    target_size = "1376x768",
    aspect_ratio = "16:9",
    transparent = false,
}
```

### 5.2 探索/搜刮背景图（bg_generic_*）

> 用于探索搜刮房间时的纸娃娃窗口背景，替代视差滚动层
> 统一参数：比例 16:9，target_size = "1032x576"，**不透明**

**构图规则**（与聚落场景一致）：
- **下方 1/3**：地面（水泥、金属板、泥地等，角色和搜刮容器站在这个区域）
- **上方 2/3**：房间内部/室外废墟环境
- 不画任何人物或容器物品，仅作为纯背景

**Prompt 模板**：
```
少女终末旅行画风，末世废土风格立绘，[场景类型描述]，
图层混合模式，手绘风格，大块色块，柔焦效果，色彩叠加，景深，
下方三分之一为地面，上方为场景环境，无任何人物，
简笔画风格，柔和线条，白色背景
```

**生成参数**：
```lua
{
    prompt = "...",
    name = "bg_generic_[场景类型]",
    target_size = "1032x576",
    aspect_ratio = "16:9",
    transparent = false,
}
```

**场景类型与房间映射**：

| scene 值 | 背景图 | 适用房间举例 |
|----------|--------|-------------|
| `ruins` | `bg_generic_ruins_industrial_*.png` | 废品堆场、印刷厂、废铁场、弹坑残骸 |
| `military` | `bg_generic_tech_facility_*.png` | 军事哨站、雷达站、军事掩体、物流中心 |
| `underground` | `bg_generic_underground_*.png` | 灌溉隧道、蘑菇洞窟、下水道深处 |
| （默认/室外） | `bg_generic_road_*.png` | 太阳能板阵列等室外场景 |

## 六、新角色 Prompt 模板

```
少女终末旅行画风，末世废土风格立绘，[性别年龄]，[名字]，[身份/职业]，[外貌特征1]，[外貌特征2]，[服装描述]，[表情/气质]，简笔画风格，柔和线条，白色背景
```

生成参数：

```lua
{
    prompt = "...",
    name = "portrait_xxx",
    target_size = "512x1024",
    aspect_ratio = "2:3",
    transparent = true,
}
```

### Prompt 编写要点

- 保持"少女终末旅行画风"前缀确保风格一致
- 外貌特征控制在 2-3 个，太多会互相干扰
- 服装是区分角色最有效的手段，优先描述
- 表情/气质用一个四字词概括即可
- 不写"末世废土"以外的背景描述，统一用"白色背景"配合透明底

# 第一章大纲：温室与废墟

---

## 一、角色参考映射

### 主角映射

| 角色 | 原型 | 核心特征 | 对应点 |
|------|------|---------|--------|
| **林砾** | 《少女终末旅行》千户 | 冷静、理性、话少、记录者、技术型、会开车 | 省略号开头、默默记录、对机械眼睛会亮、被陶夏抓到情绪流露时口是心非 |
| **陶夏** | 《少女终末旅行》尤莉 | 活泼、直觉型、好奇心强、情绪丰富 | 问很多问题、说"你每次都这样回答"、偶尔说出意外深刻的话 |

**核心互动模式：**
- 千户开车、尤莉看风景 → 林砾开车、陶夏看窗外说话
- 千户记录、尤莉问"你在写什么" → 林砾记本子、陶夏问"记什么了"
- 千户沉默、尤莉说"好想吃xxx" → 林砾沉默、陶夏说"好想走更远"

**《少女终末旅行》经典场景参考：**
```
尤莉："千户，我想吃那个。"
千户："……没有。"
尤莉："但是那边好像有。"
千户："……那边塌了。不能去。"
尤莉："你每次都这样说。"
```

对应本游戏：
```
陶夏："林砾，我们能去更远的地方吗？"
林砾："……看油够不够。"
陶夏："你每次都这样回答。"
```

### NPC参考映射

| 角色 | 参考原型 | 核心气质 | 参考理由 |
|------|---------|---------|---------|
| **沈禾** | 《银魂》登势 | 温和务实、话不多但有分量、像老妈子一样照顾人 | 经营聚落的"大姐头"气质 |
| **伍拾七** | 《星际牛仔》杰特 | 江湖气、靠谱但不正经、有原则 | 废墟营地的"老油条"气质 |

## 二、第一章核心调性

### 主题
**"两条路"** —— 在温室与废墟之间往返，熟悉两条商路，熟悉彼此

### 情感曲线
```
序章结束 → 熟悉温室 → 初识废墟 → 伍拾七的地图 → 刀鱼的画册 → 沈禾的话 → 看油够不够
（探索欲）→（适应）→（新奇）→（向往）→（温情）→（信号伏笔）→（默契）
```

### 波特风格
- 不解释林砾的信号追踪背景，只在"沈禾的话"中露出冰山一角
- NPC不主动帮忙，主角靠自己的脚跑出商路
- 日常感：往返送货、与NPC闲聊、篝火对话

## 三、章节流程

### 3.1 第三趟（SD_CH1_01）

**触发条件**：`prologue_done`，`min_trips: 2`

**场景**：行驶中/温室到废墟的路上

**核心内容**：
- 陶夏总结温室和废墟的不同"规矩"
- 路边的小树——旅程的见证者
- 林砾话变多了（被陶夏传染）

**对话要点**：
```
陶夏："温室的人买什么都砍价，但买完会说谢谢。废墟那边不砍价，但也不说谢谢。"
林砾："……各有各的规矩。就像有的松鼠过冬藏松果，有的不藏。"
```

**隐喻**：树不会走路，但一直在长——这就是它的走法

**Flag**：`sd_ch1_01_done`

---

### 3.2 外面的规矩（SD_CH1_02）

**触发条件**：`sd_ch1_01_done`，`min_trips: 3`

**场景**：废墟游民营地

**核心内容**：
- 伍拾七介绍废墟营地的规矩：不偷、不抢、不赖账
- 陶夏被调侃"话多也是一种本事"
- 林砾与伍拾七的默契（话少的人互相认可）

**对话要点**：
```
伍拾七："规矩很简单，三条：不偷，不抢，不赖账。违反任何一条——永远别来。"
陶夏："那如果卖家骗人呢？"
伍拾七："那就看你自己的眼力了。"
```

**参考场景**：《星际牛仔》杰特对斯派克说"这是原则问题"

**Flag**：`sd_ch1_02_done`，`ruins_market_intro`

---

### 3.3 伍拾七的地图（SD_CH1_02B）

**触发条件**：`ruins_market_intro`，`min_trips: 4`

**场景**：废墟游民营地

**核心内容**：
- 伍拾七展示手绘地图，北方有一片空白
- 提到"塔台"——更北边的聚落
- "空白才好，没人的路走通了就是你的"

**对话要点**：
```
伍拾七："再往北走，有个塔台。那些人怪得很，规矩比我还严。但人家有技术。"
陶夏："北边……地图上这片是空白啊。"
伍拾七："空白才好。没人的路，走通了就是你的。"
```

**伏笔**：
- 北方塔台 → 第二章目的地
- "有些地方去了就回不来了" → 远方的吸引力

**Flag**：`sd_ch1_02b_done`

---

### 3.4 画册（SD_CH1_03）

**触发条件**：`sd_ch1_02b_done`，`min_trips: 5`

**场景**：废墟游民营地角落

**核心内容**：
- 刀鱼（小姑娘）有一本灾前城市的画册
- 陶夏解释以前的城市是什么样
- 林砾用"鱼换河"的比喻解释灾变

**对话要点**：
```
刀鱼："你看这个——里面那些高楼、那些灯……以前真的长这样吗？"
陶夏："真的。以前的城市，晚上全是灯。比你看到的任何篝火都亮一百倍。"
刀鱼："那为什么没了？是不是因为太亮了，所以被收走了？"
林砾："……不是。有时候，东西不见了不是因为它太好。只是因为世界改变了。就像河改变了方向，有些鱼就得换一条河住。"
刀鱼："那鱼会难过吗？"
林砾："……大概会吧。但换了一条河之后，也许还能找到新的朋友。"
```

**参考场景**：《少女终末旅行》千户和尤莉在废弃图书馆发现旧书

**情感锚点**：
- 刀鱼是陶夏的"妹妹"型角色
- 林砾主动答应"下次来的时候讲讲以前的事"（反常）

**Flag**：`sd_ch1_03_done`

---

### 3.5 沈禾的话（SD_CH1_03B）

**触发条件**：`sd_ch1_03_done`，`min_trips: 6`

**场景**：温室社区

**核心内容**：
- 沈禾提到更东边有钟楼（第三章伏笔）
- 林砾透露收音机收到有规律的信号：三短一长
- 陶夏第一次意识到林砾"以前也收到过这种信号"

**对话要点**：
```
林砾："……收音机昨天又收到杂音了。三短一长。三短一长。"
陶夏："三短一长……那是摩尔斯电码吗？"
林砾："……也许。也许是别的东西。我记下了。"
陶夏："你以前也收到过这种信号吧？"
林砾："……偶尔。就像雨——你知道会下雨，但不知道哪一滴会先落下来。说出来也没用。但我一直在听。"
```

**揭示**：
- 林砾一直在追踪某个信号
- 她不会主动说，但被问到会承认
- 为第二章"信号来自西南方"埋伏笔

**Flag**：`sd_ch1_03b_done`，`ch1_signal_hint`

---

### 3.6 看油够不够（SD_CH1_04）

**触发条件**：`sd_ch1_03b_done`，`min_trips: 7`

**场景**：篝火

**核心内容**：
- 陶夏问"我们能跑到哪去"
- 林砾的经典回答"看油够不够"
- 林砾第一次说出不一样的话："想去哪就去哪。但先把油加满。"

**对话要点**：
```
陶夏："你说，我们能跑到哪去？"
林砾："看油够不够。"
陶夏："你每次都这样回答。"
林砾："……够的。油一直够的。我就是那个检查油的人——油不够的时候我不会出发。"
陶夏："那你以后能不能换个回答？"
林砾："……想去哪就去哪。但先把油加满。"
陶夏："你看！你说出来了！而且加了后半句更可爱！"
林砾："……别告诉别人。"
陶夏："这是我们的小秘密！"
```

**参考场景**：《少女终末旅行》篝火对话

**情感锚点**：
- 这是林砾第一次说出"想去哪就去哪"
- 两个人的"小秘密"——关系里程碑
- 从"搭档"向"默契"阶段过渡

**Flag**：`sd_ch1_04_done`，`ch1_done`

## 八、与序章的衔接

### 序章结束状态
- 两人已经去过温室和废墟
- 林砾在小本子上记录了水渠标记、蘑菇洞脚印、信号频率
- 沈禾提到"废墟有生意做"

### 第一章开始
- 从"第三趟"开始，暗示已经跑了至少两趟
- 温室和废墟已经是"熟悉的路"
- 重点是深化对两个聚落的了解

### 关系变化
| 阶段 | 序章 | 第一章结束 |
|------|------|-----------|
| 好感 | early（0-29） | early → mid 过渡 |
| 状态 | 刚开始搭档 | 有默契、有小秘密 |
| 林砾的话量 | 极少 | 开始被陶夏传染 |

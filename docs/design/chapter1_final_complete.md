# 《末世行商》第一章「熟悉的路」最终设计文档

> 融合《少女终末旅行》《奇诺之旅》《龙脉常歌》的叙事风格
> 基于现有聚落叙事系统和游戏机制

---

## 一、核心设计理念

### 1.1 叙事调性

| 参考作品 | 核心特点 | 我们的应用 |
|---------|---------|-----------|
| **少女终末旅行** | 末世中的日常诗意，螺旋循环 | 跑商本身就是节奏，不需要额外事件推动 |
| **奇诺之旅** | 单元剧结构，每个城市独立成章 | 每个聚落是一个完整小世界，有自己的规则和故事 |
| **龙脉常歌** | 情感克制深沉，日常积累终点爆发 | 关系进展靠行为变化，不靠对话表白；第一章只积累 |

### 1.2 节奏原则

- **黄金三章**：开头有氛围钩子，让玩家想知道"她们是谁、要去哪"
- **不 rushed**：给玩家沉淀情感的空间，允许"无事发生"
- **有出发感**：第一章结束时，玩家和角色一起"在路上"

### 1.3 聚落解锁规划

| 章节 | 解锁内容 | 保持新鲜感 |
|------|---------|-----------|
| 序章 | 温室社区（主聚落）+ 外围农场（前哨） | 安全区建立熟悉感 |
| 第一章 | 废墟营地（首个外部聚落）+ 周边资源点 | "外面"的初次体验 |
| 第二章 | 北穹塔台 + 钟楼书院 | 全新类型的聚落 |
| 第三章 | 铁锁商队（势力登场） | 新势力而非新聚落 |

## 二、序章「第一道车辙」（3-5分钟）

### 2.1 开场钩子（氛围而非事件）

**画面**（15秒）：
```
灰色天空。废弃公路。
一辆货车停在路边，引擎盖打开，冒着淡淡的白烟。

林砾的手上沾着机油，正在检查发动机。
陶夏坐在副驾驶座上，腿晃着，看着远处。

没有人说话。
```

**钩子**：车坏了？她们是谁？要去哪？

### 2.2 第一次互动（30秒）

```
陶夏："还能修好吗？"

林砾："能。"

陶夏："你上次也是这么说的。然后我们在路上停了四个小时。"

林砾："……这次不一样。"

陶夏叹了口气，但没说什么。她从座位下摸出一个苹果，
在衣服上擦了擦，开始吃。
```

**钩子**：两人的关系？是搭档还是朋友？为什么在路上？

### 2.3 玩家第一次选择（1分钟）

修好后，陶夏问："接下来往哪走？"

| 选项 | 结果 |
|------|------|
| "回温室" | 林砾："油够。可以再往前走走。"（车继续往前） |
| "往前走走看" | 林砾："嗯。"（车继续往前） |

**设计**：无论选什么，车都会往前开。但选择本身让玩家参与了"出发"的决定。

### 2.4 第一次驾驶（1分钟）

不是伏击，而是氛围变化：

```
路两边的风景开始变化。
温室的绿色渐渐退去，露出灰色的废土。

陶夏突然说："你有没有觉得……"
林砾："嗯？"
陶夏："……没什么。"

她没说出口的话，消散在引擎声里。
```

**钩子**：她想说什么？为什么没说？

### 2.5 到达温室社区（30秒）

沈禾出来迎接，不是隆重欢迎：

> "哦，是你们俩。这次带了什么？"

**钩子**：她们来过很多次？这是日常？那这次有什么不一样？

### 2.6 序章结束

**篝火场景**：
```
夜里。温室外的空地。

陶夏："今天路上……你本来想问什么？"
林砾："忘了。"
陶夏："骗人。"
林砾："……真的忘了。"

沉默。篝火噼啪作响。

陶夏："下次记得告诉我。"
林砾没说话，但往火堆里添了根柴。
```

**设置 flag**：`prologue_done`

## 三、第一章「熟悉的路」

### 3.1 触发条件

- `prologue_done`
- 完成 ≥ 3 趟跑商
- 访问过外围农场

### 3.2 第一章前半：建立期（5-8趟跑商）

**活动范围**：温室社区 ↔ 外围农场（熟路）

**核心体验**：
- 建立"跑商日常"的节奏感
- 通过多次互动逐步展示两人关系
- 让玩家熟悉游戏系统

**新增内容**：

| 系统 | 内容 |
|------|------|
| **闲逛** | 温室社区 2-3 个场景（浇水、育种棚、老甘的故事） |
| **篝火** | 3-4 次对话，"无事发生"为主 |
| **Chatter** | 第一章专属日常对话 |
| **Radio** | 鸣砂广播开始提及"远方有聚落" |

**关系进展（行为变化而非对话）**：

| 阶段 | 表现 |
|------|------|
| 早期 | 陶夏主动找话题，林砾简短回应 |
| 中期 | 陶夏不用说完整句话，林砾就知道她要什么 |
| 后期 | 沉默也不尴尬，两个人各做各的事 |

### 3.3 第一章中点：余韵事件

**触发**：第 5-6 趟跑商后，首次到达废墟营地

**场景：刀鱼的画册**

```
废墟营地。少女拾荒者刀鱼拉住你的袖子。

刀鱼："姐姐，你们跑过很多地方吧？"
陶夏："嗯。"

刀鱼翻出一本旧画册，封面掉了一半。
里面是灾前城市的照片——高楼、商场、公园。

刀鱼："这些地方……真的存在过吗？"
陶夏看了很久。

陶夏："存在过。"
刀鱼："那为什么没了？"

陶夏不知道怎么回答。

林砾走过来，看了一眼画册。
林砾："下次来的时候，我给你讲讲。"

刀鱼眼睛亮了。
```

**设计意图**：
- 安静的一瞬，留下印象但不推动剧情
- 刀鱼代表"没见过旧世界的孩子"
- 林砾主动开口——关系进展的体现

### 3.4 第一章后半：展开期（3-5趟跑商）

**活动范围**：温室社区 ↔ 废墟营地（新路已熟悉）

**核心变化**：
- 陶夏开始主动接"更远"的单子
- 林砾不再反对"冒险"
- 收音机信号出现（林砾会在某个频率停留很久）

**关键场景**：

**场景 A：沈禾的随口一提**
```
沈禾："东边还有别的聚落，但没去过。路太远了，
       我们的菜送不到那么远。"
陶夏："别的聚落？"
沈禾："听说有个塔台，还有群读书人住在钟楼里。
       但只是听说。"

陶夏看了林砾一眼。林砾正在检查轮胎，
但她的动作慢了一拍——她在听。
```

**场景 B：伍拾七的地图**
```
伍拾七："你们跑的路线越来越远了。"
陶夏："想多看看。"
伍拾七笑了笑，在地图上画了一条线。
伍拾七："再往前，有个塔台。那帮人怪得很，
       但规矩比我们还严。"

他把地图推过来。
伍拾七："想去就去。但记住——
       有些地方，去了就回不来了。"

他说的是警告，也是祝福。
```

### 3.5 第一章结束：出发

**触发条件**：
- 完成 ≥ 8 趟跑商
- 温室好感 ≥ Lv1（20）
- 废墟好感 ≥ Lv1（20）

**最终场景：篝火**

```
夜里。废墟营地外。

两个人坐在货车旁边，看着远处的路。
那条路往东南方向去，消失在灰色的地平线。

陶夏："你说，我们能跑到哪？"

林砾："看油够不够。"

陶夏笑了："你每次都这样回答。"

林砾没说话。她翻开记录本，
在最新一页写了个数字——
不是油量，是某个频率。

陶夏看见了，但没问。

沉默了一会儿。

陶夏："明天往哪走？"
林砾合上本子。
林砾："……往前。"

她没有说"前面是哪"。
但陶夏知道。她也知道林砾知道她知道。

这就够了。
```

**结果**：
- 车往东南方向开去
- 解锁新路线（通往塔台/书院方向）
- 玩家和角色一起：我们出发了

**设置 flag**：`ch1_done`

## 四、情感弧线设计

### 4.1 序章→第一章的情感走向

```
序章：钩子（好奇）
    ↓
第一章前半：日常（熟悉）
    ↓
第一章中点：余韵（印象）
    ↓
第一章后半：展开（期待）
    ↓
第一章结束：出发（决心）
```

### 4.2 关系进展（克制式）

| 阶段 | 对话表现 | 行为表现 |
|------|---------|---------|
| 序章 | 客气，找话题 | 分工明确，但有点僵硬 |
| Ch1早期 | 开始吐槽对方 | 默契出现，但不说破 |
| Ch1中期 | 沉默也不尴尬 | 一个眼神就知道要做什么 |
| Ch1后期 | 不用说完整句话 | 为对方考虑，但不承认 |
| Ch1结束 | "往前" | 共同面对未知 |

### 4.3 玩家情感体验

| 时间点 | 玩家感受 |
|--------|---------|
| 序章结束 | "她们是谁？想去哪？" |
| Ch1前半 | "跑商还挺舒服的，两个人挺有意思" |
| Ch1中点 | "刀鱼那幕挺有感觉的……" |
| Ch1后半 | "前面还有东西，想继续玩" |
| Ch1结束 | "我们出发了" |

## 五、与游戏机制的结合

### 5.1 教学融入叙事

| 教学内容 | 叙事包装 |
|---------|---------|
| 接单 | 沈禾："老规矩，两箱罐头" |
| 驾驶 | 从温室到农场，熟悉的风景 |
| 交易 | 陶夏砍价，林砾在旁边摇头 |
| 篝火 | 两个人第一次过夜 |
| 探索 | 刀鱼的画册场景 |

### 5.2 各叙事通道职责

| 通道 | 第一章内容 |
|------|-----------|
| **主线对话** | 2-3 个，聚落领袖互动 |
| **篝火对话** | 6-8 个，日常闲聊为主 |
| **NPC对话** | 沈禾、赵苗、伍拾七、刀鱼 |
| **到处晃晃** | 温室 3 个，废墟 3 个 |
| **随机事件** | 路点事件，不触发战斗 |
| **Chatter/Radio** | 第一章专属内容 |

### 5.3 悬念埋设（为后续章节）

| 悬念 | 埋设方式 | 后续回收 |
|------|---------|---------|
| 收音机信号 | 林砾偶尔停留某个频率 | Ch2 揭示她在听什么 |
| 远方聚落 | 沈禾、伍拾七随口提及 | Ch2 解锁塔台/书院 |
| 铁锁商队 | 废墟传闻"北边有大商队" | Ch3 正面遭遇 |
| 灾前世界 | 刀鱼的画册 | 全书世界观铺垫 |

## 六、与聚落叙事的衔接

### 6.1 第一章解锁的聚落内容

**温室社区**：
- 闲逛：浇水、育种棚、老甘的故事
- 好感 Lv1：沈禾开始委托私事（水泵零件）

**外围农场**：
- 赵苗的委托收获系统开放
- 熟悉前哨站概念

**废墟营地**：
- 闲逛：黑市、刀鱼、拾荒者
- 好感 Lv1：伍拾七认可，黑市系统开放
- 刀鱼成为可交互 NPC

### 6.2 为后续章节预留

**北穹塔台**（Ch2 解锁）：
- 技术派，完全不同的氛围
- 信号、数据、远方
- 韩策、纪微登场

**钟楼书院**（Ch2 解锁）：
- 学者派，历史感
- 灾前记忆、文明传承
- 白述、谢令登场

**铁锁商队**（Ch3 登场）：
- 不是聚落，是势力
- 姜寒、小铆登场
- 经济压力、理念冲突

## 七、检验清单

### 7.1 节奏检验

- [ ] 序章 3-5 分钟，有钩子但不 rushed
- [ ] 第一章 30-60 分钟，允许"无事发生"
- [ ] 结尾有"出发"感，但不靠大事件
- [ ] 四个聚落没有全亮相，后面还有新鲜感

### 7.2 情感检验

- [ ] 关系进展靠行为变化，不靠对话表白
- [ ] 玩家对两人关系产生好奇
- [ ] 玩家想继续玩下去"看看会发生什么"
- [ ] 第一章结束时，玩家和角色一起"在路上"

### 7.3 机制检验

- [ ] 教学融入叙事，不突兀
- [ ] 各叙事通道职责清晰
- [ ] 悬念为后续章节铺垫
- [ ] 与聚落叙事系统兼容

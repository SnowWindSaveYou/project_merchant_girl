## 一、第二章核心调性

### 主题
**"更大的世界"** —— 探索温室和废墟之外的区域，第一次跑通完整商路

### 情感曲线
```
出发 → 探路（未知）→ 到达（挫折：韩策冷淡）→ 交易 → 发现（信号）→ 结章
（向往）→（紧张）→（被审视）→（认可）→（共鸣）→（成长）
```

### 与第一章的差异
| 维度 | 第一章 | 第二章 |
|------|--------|--------|
| 范围 | 温室↔废墟（熟悉） | 向北探索（未知） |
| 节奏 | 往返送货、日常 | 探路、新聚落、新NPC |
| NPC | 沈禾、伍拾七、刀鱼 | 韩策、白述、小铆 |
| 林砾 | 被陶夏"传染"话变多 | 遇到同类（技术宅）、话更多 |
| 信号 | 三短一长伏笔 | 确认来自西南方 |

### 新增NPC映射

| 角色 | 参考原型 | 核心气质 | 参考理由 |
|------|---------|---------|---------|
| **韩策** | 《命运石之门》冈部伦太郎 | 傲慢、技术狂、嘴硬心软、说话中二 | 塔台技术派的"科学家"气质 |
| **白述** | 《虫师》银古 | 温和淡然、见过很多、不评判、只是记录 | 学者的"旁观者"气质 |
| **小铆** | 《天元突破》西蒙（后期） | 害羞但技术强、眼睛会亮 | 技术宅的"惺惺相惜"气质 |

## 二、章节流程

### 2.1 出发（SD_CH2_01）

**触发条件**：`ch1_done`

**场景**：温室/废墟首页

**核心内容**：
- 陶夏提议往北走——伍拾七提过的塔台
- 林砾同意（呼应第一章"想去哪就去哪"）
- "空白才好"——伍拾七的话在两人心里种下了种子

**对话要点**：
```
陶夏："伍拾七说的那个塔台——要不我们去看看？"
林砾："……中间那段路没人走过。"
陶夏："那就我们第一个走。"
林砾："……好。"
```

**Flag**：`ch2_started`

---

### 2.2 北穹塔台（SD_CH2_02）

**触发条件**：`ch2_started`，第一次到达塔台

**场景**：塔台节点

**核心内容**：
- 韩策的冷淡接待、严格的入关检查
- 陶夏被怼但不记仇
- 林砾注意到塔台的通信设备——比信号中继站先进

**对话要点**：
```
韩策："身份登记，货物清单，行驶路线。三样齐了再进。"
陶夏："好凶……"
韩策："规矩是死的，但能让活人安心。"
林砾："……理解。"
```

**韩策态度变化**：
- 初见：冷淡审视
- 交易后：点头认可（"你们的车还行"）
- 聊技术时：话变多

**Flag**：`npc_han_ce_intro`，`tutorial_arrived_tower`

---

### 2.3 技术交流（SD_CH2_03）

**触发条件**：`tutorial_arrived_tower`

**场景**：塔台工坊

**核心内容**：
- 小铆是塔台的年轻技工，与林砾年龄相仿
- 两人聊引擎、聊改装——眼睛都亮了
- 陶夏吐槽"你们两个话都变多了"

**反差萌展示**：
```
陶夏："林砾你刚才说了整整三句话！而且都有专业术语！"
林砾："……没有。只是正常的交流。"
陶夏："你的眼睛在亮！"
林砾："……反射的火光。"
```

**情感锚点**：林砾遇到了"同类"

---

### 2.4 档案室（SD_CH2_04）

**触发条件**：`npc_han_ce_intro`

**场景**：塔台档案室

**核心内容**：
- 白述是档案管理员，收藏旧世界资料
- 林砾询问关于旧信号塔的记录
- 揭示：灾前有七个中继站，西南方据说还有一个

**对话要点**：
```
白述："信号塔？灾前有七个中继站。现在大多塌了。"
林砾："……西南方那个还能用吗？"
白述："据说。但没人验证过。"
陶夏："那我们去看看不就知道了？"
```

**Flag**：`npc_bai_shu_intro`

---

### 2.5 信号发现（SD_CH2_05）

**触发条件**：`npc_bai_shu_intro` + 夜晚篝火

**场景**：塔台附近/篝火

**核心内容**：
- 林砾检查塔台通信设备，确认信号来自西南方
- 韩策发现她的行为——透露塔台也在追踪
- 两人达成某种默契

**对话要点**：
```
韩策："西南方的信号？我们也在追踪。但太远了，油不够。"
林砾："……油够的话，你们会去？"
韩策："也许。但塔台需要人守着。你们自由。想去就去。"
```

**揭示**：
- 林砾的追踪不是孤独的——有人在听同样的东西
- 塔台的通信设备比信号中继站先进得多
- 韩策从冷淡到"你们自由"

---

### 2.6 结章（SD_CH2_06）

**触发条件**：完成塔台主要任务

**场景**：篝火/麻薯号

**核心内容**：
- 回顾这一趟：温室 → 废墟 → 塔台
- 商路跑通了，世界变大了
- 林砾收起本子——西南方

**对话要点**：
```
陶夏："我们走过的路好像变长了好多。"
林砾："……嗯。还会更长。"
陶夏："油够吗？"
林砾："够的。一直够的。"
```

**Flag**：`ch2_done`

## 七、CG需求表（序章~第二章）

> CG数量控制在精简范围，只保留最具叙事价值的场景

| 章节 | 场景 | CG描述 | 触发时机 | 优先级 |
|------|------|--------|---------|--------|
| 序章 | 麻薯号命名 | 陶夏兴奋地指着车，林砾低头看本子，车在夕阳下 | SD_TUTORIAL_FIRST_DEPARTURE 结束 | 高 |
| 序章 | 温室初见 | 玻璃穹顶下，绿色的光，沈禾站在菜地里 | SD_TUTORIAL_GREENHOUSE_ARRIVAL | 高 |
| 第一章 | 画册 | 三人蹲在碎砖旁，刀鱼捧着画册，陶夏和林砾一起看 | SD_CH1_03 | 中 |
| 第一章 | 篝火秘密 | 夜晚篝火旁，两人对视后别过头，火光映在脸上 | SD_CH1_04 结束 | 高 |
| 第二章 | 北穹塔台 | 铁灰色的建筑群，韩策站在门口检查清单 | SD_CH2_02 | 中 |
| 第二章 | 技术交流 | 林砾和小铆蹲在引擎旁，两人眼睛都亮着，陶夏在后面吐槽 | SD_CH2_03 | 低 |

**总计：6张CG**

### CG优先级说明
- **高**：关键情感节点，必须有
- **中**：场景氛围强化，建议有
- **低**：角色互动点缀，可省略

### CG风格要求
- 波特绘本风格：柔和的光、自然的细节、像水彩画
- 不追求"帅气"或"精细"，而是"温暖"和"故事感"
- 人物表情要细腻，尤其是林砾的"口是心非"瞬间

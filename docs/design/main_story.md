### 比阿特丽克斯·波特风格原则

> 温和叙述、自然隐喻、"不解释"、"沉默的观察者"

1. **不解释角色的动机**——让行为自己说话
2. **用自然意象代替心理描写**——"风吹过麦田"比"她感到惆怅"更好
3. **对话留白**——省略号、沉默、未说完的话比长篇解释更有力量
4. **日常感优于戏剧化**——聚落的问候、路边的树、篝火上的汤，这些都是叙事
5. **沉默的观察者**——NPC不主动帮忙，不急于解释世界，像山一样只是存在

## 一、设计理念

### 核心叙事主题

> **"在一个断裂的世界里，连接本身就是一种勇气。"**

主线围绕一个问题展开：**被切断的世界，值得被重新连在一起吗？**

- 主角用贸易回答"值得"
- 铁锁商队用垄断回答"只能由我们来连"
- 静潮社用沉默回答"不值得，断开才安全"
- 最终答案由玩家的行为——而非一次选择——来给出

### 叙事调性

- **温暖治愈为底色**，允许出现紧张和冲突，但不走向绝望或残酷
- 反派不是"坏人"，而是"用另一种方式回应恐惧的人"
- 每个冲突都有和解的可能，但和解需要代价（时间、资源、信任）
- 最终所有结局都是积极的，只是"积极的方式"不同

### 节奏原则

- **慢热型叙事**：主线不催促玩家，而是跟随跑商节奏自然展开
- 每章之间鼓励玩家自由跑商 5-15 趟，积累好感和资源
- 主线推进靠"里程碑触发"，不靠限时任务
- 主线事件穿插在日常跑商之中，让玩家感觉"故事是自然发生的"

## 二、新势力设定

### 势力 A：铁锁商队

| 项目 | 设定 |
|------|------|
| **定位** | 废土上最大的有组织商队，控制着多条主干贸易路线 |
| **理念** | "自由贸易是混乱之源，只有统一管控的贸易才能避免流血" |
| **组织形态** | 3-5 辆重型改装货车编队行动，有护卫、有调度、有定价权 |
| **势力范围** | 主要控制北穹塔台—废墟营地之间的东线商路，以及部分远距离路线 |
| **与主角的关系** | 起初是竞争对手——压价、抢单、封路；中期是谈判对象；后期可能成为盟友 |
| **内部矛盾** | 铁锁内部并非铁板一块：领袖姜寒坚持秩序，年轻成员小铆更向往自由 |
| **设计意图** | 代表"善意的控制"——她们不是劫匪，确实在维持废土贸易秩序，只是方式是垄断 |

**铁锁商队与聚落的关系**：

| 聚落 | 关系 | 说明 |
|------|------|------|
| 北穹塔台 | 主要客户 | 韩策默许铁锁垄断部分技术品流通，换取稳定的工业品供应 |
| 温室社区 | 被动依赖 | 沈禾不喜欢铁锁的高价，但冬季只有铁锁愿意送货 |
| 废墟游民营地 | 对抗 | 伍拾七视铁锁为眼中钉，拒绝接受统一定价 |
| 钟楼书院 | 冷淡 | 白述认为铁锁只关心利润，不关心文明传承 |

### 势力 B：静潮社

| 项目 | 设定 |
|------|------|
| **定位** | 分散在废土各处的松散组织，信奉"断联即安全" |
| **理念** | "旧世界因为过度连接而崩溃，每个聚落应该自给自足，不再互相依赖" |
| **组织形态** | 没有统一指挥，各地有独立的"守静人"，通过留言和暗号松散联络（讽刺的是，他们自己也在"连接"） |
| **行为模式** | 拆除路标、堵塞废弃隧道入口、在聚落散布"贸易带来掠夺者"的言论、偶尔破坏信号中继设备 |
| **与主角的关系** | 起初是无形的阻碍（路标被挪、隧道被封），中期直接对话产生冲突，后期有和解空间 |
| **核心悲剧** | 静潮社的创始者们曾亲眼见证：某个聚落因为贸易纷争引来掠夺者而被毁灭。他们的恐惧是真实的 |
| **设计意图** | 代表"恐惧驱动的善意"——他们不是在害人，而是在用错误的方式保护人 |

**静潮社不是统一的敌人**：

- 有人只是默默拆路标，因为怕商队带来掠夺者
- 有人是激进分子，会主动破坏信号设备
- 有人内心动摇，看到主角的善意贸易后开始怀疑自己的信念
- 最终不需要"击败"静潮社，而是让他们看到"连接也可以不重蹈覆辙"

## 三、新 NPC 设定

### 铁锁商队 NPC

#### 1. 姜寒（铁锁商队队长）

| 项目 | 设定 |
|------|------|
| **年龄区间** | 38–42 岁 |
| **外观关键词** | 束发、旧军大衣改的商队外套、皮带上挂计算器和短波对讲机、左手无名指缺失一节 |
| **性格核心** | 冷静果断，信奉"规则比善意可靠"，但对手下有母亲般的保护欲 |
| **过去经历** | 年轻时也是自由行商，搭档在一次贸易纷争中被杀。从此认为只有垄断定价才能防止暴力 |
| **对主角态度** | 初期鄙视（"两个小姑娘能活多久？"），中期尊重（"你们居然还没死"），后期欣赏但仍然固执 |
| **转变关键** | 当她看到主角不靠垄断也能让聚落和平交易时，她的信念开始松动。但真正让她改变的，是小铆说"我不想变成一个被人害怕的商人" |
| **功能作用** | 中期主要对抗源，后期可转化为盟友，提供编队护送、远距离路线信息 |

**立绘描述**：
- 正向：38-42 岁女性，东亚面孔，身形结实但不粗壮，气质严厉而疲惫；深棕色短发后束成利落低马尾；深褐色眼睛，常半眯着打量人；穿改良军大衣式商队外套，内搭高领毛衫，腰间皮带挂旧计算器和金属水壶；左手戴半指手套，无名指短一截；站姿双臂抱胸或单手叉腰，像在做决定。
- 反向约束：避免女将军感、避免魅惑风、避免华丽饰品、避免少女化。

#### 2. 小铆（铁锁商队年轻机械师）

| 项目 | 设定 |
|------|------|
| **年龄区间** | 16–18 岁 |
| **外观关键词** | 油渍工装裤、绑在额头的旧焊接护目镜、总在口袋里掏零件、笑起来露虎牙 |
| **性格核心** | 好奇心旺盛，技术天赋极高，内心向往自由但又害怕背叛姜寒 |
| **与林砾的关系** | 技术上的惺惺相惜——两人聊起引擎和零件时眼睛都会亮。林砾是小铆第一个在铁锁之外交到的朋友 |
| **功能作用** | 铁锁内部的"裂缝"，为主角提供铁锁内部信息；后期可能脱离铁锁加入贸易联盟 |
| **剧情价值** | 代表"下一代不必重复上一代的恐惧" |

**立绘描述**：
- 正向：16-18 岁少女，短发蓬松微卷，额头绑旧护目镜当发箍；穿宽大工装裤、打了补丁的背心、厚底旧运动鞋；口袋鼓鼓囊囊塞满螺丝和小工具；手上有机油污渍但不脏兮兮而是"正在工作中"的状态；表情开朗好奇，笑起来有虎牙。
- 反向约束：避免性感化、避免偶像系、避免军装化。

### 静潮社 NPC

#### 3. 陆沉（静潮社守静人代表）

| 项目 | 设定 |
|------|------|
| **年龄区间** | 48–55 岁 |
| **外观关键词** | 灰白短发、旧棉服、围巾遮半张脸、手持旧步话机（关着的）、眼神疲惫但清醒 |
| **性格核心** | 温和但坚定，说话慢条斯理，从不提高声音，但也几乎不让步 |
| **过去经历** | 曾是某个繁荣贸易聚落的管理者。一次大规模交易引来掠夺者，聚落被毁，他的家人在混乱中失散。从此成为"断联主义"的践行者 |
| **对主角态度** | 不敌视，甚至有些心疼（"你们还这么年轻，不要重蹈覆辙"）。他反对的不是两个女孩，而是"连接"这个行为本身 |
| **转变关键** | 当主角帮助一个聚落抵御危机——不是靠武力，而是靠贸易网络的互助——他开始动摇 |
| **功能作用** | Ch.5 的核心对话对象，后期可提供被封锁的旧路线信息作为和解礼物 |

**立绘描述**：
- 正向：48-55 岁中年男性，面容憔悴但不凶狠，灰白短发，眼下有深重黑眼圈；穿旧棉袄外套，领口围暗色围巾遮住半张脸；手里总拿着一部关着的旧步话机，像某种纪念物；坐姿或靠墙站立，不像在警戒，更像在等什么人。
- 反向约束：避免反派化、避免武装感、避免宗教领袖感。

#### 4. 霜期（静潮社脱离者）

| 项目 | 设定 |
|------|------|
| **年龄区间** | 19–22 岁 |
| **外观关键词** | 灰绿斗篷、布缠短刀、写满笔记的手臂绷带、眼神警觉但不冷漠 |
| **性格核心** | 在静潮社长大，内心认同"保护"的出发点，但逐渐怀疑"断联"是否真的有效 |
| **与陶夏的关系** | 年龄相近但成长环境截然相反——陶夏在流动中长大相信连接，霜期在封闭中长大被教导隔离。两人的对话是主线中最有思辨张力的段落 |
| **功能作用** | Ch.4-5 期间出现的流动 NPC，提供静潮社内部视角，后期成为两方沟通的桥梁 |
| **剧情价值** | 代表"在恐惧中长大的人也可以选择信任" |

**立绘描述**：
- 正向：19-22 岁年轻女性，身形修长警觉；灰绿色连帽短斗篷，内搭紧身防风衣；短发利落，一侧别着旧金属发夹；左小臂缠绷带，上面密密麻麻写着笔记（她的习惯）；腰间别着布缠短刀但很少拔出；眼神有戒心但藏着好奇。
- 反向约束：避免杀手/刺客化、避免暴露装、避免可爱化。

### 新增聚落 NPC（补充现有聚落厚度）

#### 5. 纪微（北穹塔台 / 信号技术员）

| 项目 | 设定 |
|------|------|
| **年龄区间** | 22–25 岁 |

#### 6. 老甘（温室社区 / 育种老人）

| 项目 | 设定 |
|------|------|
| **年龄区间** | 65–70 岁 |
| **定位** | 温室社区最年长的居民，灾前农学院的学生 |
| **性格** | 慈祥、唠叨、记忆力惊人，总在讲灾前的故事 |

#### 7. 刀鱼（废墟游民营地 / 少女拾荒者）

| 项目 | 设定 |
|------|------|
| **年龄区间** | 13–15 岁 |
| **外观关键词** | 乱蓬蓬的短发、晒黑的皮肤、打了很多补丁的旧马甲、塞满杂物的军用挎包、脖子上挂着一个旧指南针当项链 |
| **性格核心** | 表面油嘴滑舌、爱吹牛，内心极度渴望被认可。总想证明自己"不比大人差"，但被夸的时候反而会手足无措 |
| **过去经历** | 不记得父母，从记事起就在废墟里翻东西。被伍拾七捡回营地前，独自在外围废墟活了至少两年。伍拾七是她事实上的监护人，但她绝不会承认 |
| **与主角的关系** | 对陶夏有强烈的"姐姐崇拜"——觉得陶夏又会说话又会砍价还到处跑过，是她想成为的那种人。对林砾则是安静的信任——林砾答应"下次给你讲讲"城市的事，她记得很牢 |
| **与伍拾七的关系** | 嘴上叫"老大"，心里当家长。伍拾七对刀鱼是整个营地里最没脾气的时候——虽然嘴硬说"这丫头烦死了" |
| **定位** | 伍拾七手下最小的拾荒者，机灵胆大 |
| **功能作用** | 废墟线的情感锚点，让拾荒帮从"灰色势力"变成"有孩子的家"；黑市的非官方向导；旧时代物品的鉴定者（虽然经常鉴定错） |
| **剧情钩子** | 她偷偷收藏了一本画册，里面画的是灾前城市的样子——但她从没见过真正的城市 |
| **剧情价值** | 代表"在废墟里长大的孩子，依然可以对世界充满好奇"。她不悲惨——她很快乐，只是那种快乐建立在不知道自己失去了什么的基础上 |

**功能 NPC 角色**：

| 场景 | 角色 | 说明 |
|------|------|------|
| 黑市解锁 | 向导 | 带陶夏逛市场、介绍各个摊位，比伍拾七的规矩宣讲更生动 |
| 废墟营地驻留 | 可对话 NPC | 到达营地时可以找她聊天，对话内容随好感变化 |
| 黑市淘货 | 彩蛋供货 | 她的摊位偶尔出现"刀鱼精选"——她认为很值钱的旧时代小物件（可能是垃圾，也可能真的值钱） |
| 旅途偶遇 | 随机事件 | 在废墟营地附近路线上偶尔遇到她在外面翻废墟，可以交换物品或听她吹牛 |

**立绘描述**：

### 第三章：铁锁

| 项目 | 内容 |
|------|------|
| **触发条件** | `ch2_done` + 完成 ≥ 15 趟跑商 + 任一聚落好感 ≥ Lv1(30) |
| **核心目标** | 正面遭遇铁锁商队，感受"垄断贸易"的压力 |
| **新要素** | 姜寒登场；小铆加入铁锁；经济压力机制 |
| **情感基调** | 紧张、不服气、隐约敬佩 |

**叙事目的**：
- 引入竞争对手，让玩家体验"被挤压"的感觉
- 展示另一种"连接世界"的方式——垄断
- 小铆的选择：从塔台到铁锁，展示年轻人的抉择
- 让主角开始思考：仅靠善意跑商，够吗？

**剧情要点**：

1. **铁锁出现**：到达某个聚落时，发现铁锁商队的车队已经停在门口。
   - 三辆重型改装货车排成一列，每辆都比主角的车大两倍
   - 姜寒靠在车门上打量两人："又是散户。现在还有人跑单车？"
   - 陶夏立刻不爽。林砾按住她。

2. **小铆的转变**：让玩家惊讶的是——小铆在铁锁车队里。
   - 陶夏："等等，你不是塔台的那个……"
   - 小铆："我上个月加入铁锁了。他们有更好的设备，能跑更远的路。"
   - 林砾沉默。小铆偷偷说："队长说你们迟早会被淘汰。但我觉得……你们的车修得挺好的。"
   - **设计意图**：展示年轻人的选择——不是背叛，只是另一种路

3. **经济压力**：铁锁商队在本章期间对部分商品执行"控价"——
   - 若玩家去某个聚落卖工业品，发现铁锁已经压低了收购价
   - 3-5 趟跑商期间会感受到利润下降
   - **设计意图**：让玩家真正感到"被挤压"的不适，但不至于无法生存

4. **姜寒的条件**：姜寒主动找到主角，提出合并邀请——
   - "加入铁锁。你负责维修，你负责谈判。工资稳定，路上安全，不用担心下一顿。"
   - **选择 A**："我们考虑一下"（获得 3 趟跑商的价格保护，但后续剧情延迟）
   - **选择 B**："谢谢，但我们想自己跑"（姜寒点头，"那就别怪市场不留情"）
   - **选择 C**（陶夏抢答）："我们的车虽然小，但跑得到你们去不了的地方"（姜寒笑了，"有意思"）
   - 无论选什么，最终都会拒绝——因为这是主角的核心价值观

5. **章节高潮**：一次跑商途中，主角发现某条常走的路被铁锁设了"收费关卡"——
   - 不是武力封锁，而是"我们维护了这段路，通行请付维护费"
   - 陶夏暴怒。林砾安静地算了算账，说"绕路更贵"。
   - 最终选择：付费通过 / 绕路 / 找伍拾七帮忙走地下通道

6. **章节收束**：夜晚篝火——
   - 陶夏："她说得对吗？我们迟早会被淘汰？"
   - 林砾："……她说的那种'安全'，不是我们想要的。"
   - 陶夏："那我们想要什么？"
   - 林砾没有回答。但第二天早上，她在记录本上多写了一行字：**"让更多人能自己选择跟谁交易。"**

**铁锁商队主题事件**：

| EVT_101 | 被抢走的订单 | trade | 到达聚落时发现高利润订单已被铁锁接走 |
| EVT_102 | 铁锁的善意 | encounter | 路遇铁锁护卫队，他们主动帮你驱走了掠夺者（并顺便宣传"加入我们更安全"） |

**解锁**：
- 铁锁相关地图标记出现
- 伍拾七好感需求下降（共同敌人拉近距离）

**设置 flag**：`ch3_done`、`iron_lock_encountered`、`xiaomao_iron_lock`、`jiang_han_offer_declined`

### 第四章：裂痕与种子

| 项目 | 内容 |
|------|------|
| **触发条件** | `ch3_done` + 完成 ≥ 25 趟跑商 + 任一聚落好感 ≥ Lv2(60) |
| **核心目标** | 聚落之间爆发冲突，主角必须做出艰难选择 |
| **新要素** | 温室—塔台冲突、林砾与陶夏的分歧、霜期初次登场 |
| **情感基调** | 焦虑、责任感、痛苦但不绝望 |

**叙事目的**：
- 展示聚落之间的真实矛盾
- 让玩家体验"夹在中间"的困境
- 林砾与陶夏的分歧——两人价值观的微妙差异
- 霜期出现，为静潮社埋线

**剧情要点**：

1. **危机爆发**：
   - 温室社区的水循环系统出了问题（与 `Q_SHEN_HE_SEEDS` 相关），急需塔台的水泵控制模块
   - 韩策以此为筹码，要求温室签"长期独占食物供应协议"
   - 沈禾拒绝："一个聚落的命脉不能交到另一个聚落手里"
   - 两个聚落暂停了彼此的贸易——主角无法在双方之间接单

2. **林砾与陶夏的分歧**：
   - **林砾**倾向维持中立："这是他们之间的事。我们插手可能让情况更糟。"
   - **陶夏**倾向介入："沈禾明明在求助！我们跑商不就是为了连接人吗？如果连这种时候都不帮——"
   - 不是吵架，而是沉默的拉扯——林砾做饭时不说话，陶夏睡觉时背对着她
   - **设计意图**：让玩家感受到"即使是最亲密的搭档，面对真实困境也会产生裂痕"

3. **玩家行动**（三条路径，不互斥，可组合）：
   - **路径 A：说服韩策让步** — 需要塔台好感 ≥ Lv2，或完成纪微的请求（帮她修复一台旧设备），韩策会降低条件
   - **路径 B：帮温室自救** — 从废墟营地或黑市找到替代零件，虽然性能差一些，但解了燃眉之急
   - **路径 C：让铁锁居中** — 联系姜寒做中间人，铁锁可以调停，但会趁机扩大对两个聚落的影响力
   - 每条路径都有代价，没有完美解法

4. **霜期的登场**：
   - 在主角处理聚落冲突期间，某次旅途中遇到一个陌生女孩站在被拆除的路标旁
   - "你们是行商吧。"她没有敌意，只是观察。"有人说你们能解决聚落的问题。我不太信。"
   - 她留下一句话就消失了："连自己聚落的事都解决不了，还想把更多聚落连在一起？"
   - 不点明她是静潮社的人，留下悬念

5. **裂痕修复**：无论玩家选了哪条路径，最终都会解决水泵危机（只是代价不同）。当温室重新运转那天——
   - 沈禾站在温室门口叹气："我一直觉得，只要守好自己的地方就够了。但如果没有你们跑来跑去……"
   - 韩策不会道歉，但会默默降低下一批货的报价
   - **设计意图**：冲突解决后的温暖，比没有冲突的温暖更有重量

6. **林砾与陶夏和好**：
   - 不是正式的"和好"，而是——
   - 陶夏在货车引擎盖上敲了两下（林砾正在检修）："喂。还生气吗？"
   - 林砾没抬头："我没生气。我在想你说得对不对。"
   - 陶夏："那你想出来了吗？"
   - 林砾："……你说得不全对。但我们确实不能什么都不做。"
   - 陶夏笑了："够了够了。走吧，还有一单要送。"

**解锁**：
- 温室—塔台贸易恢复，部分商品价格优化
- 霜期成为流动 NPC（在不同节点随机出现）
- 纪微作为独立 NPC 可交互

**设置 flag**：`ch4_done`、`crisis_resolved`、`shuangqi_met`、`linli_taoxia_reconciled`

### 第五章：静潮

| 项目 | 内容 |
|------|------|
| **触发条件** | `ch4_done` + 完成 ≥ 35 趟跑商 + ≥ 2 个聚落好感 ≥ Lv2(60) + `shuangqi_met` |
| **核心目标** | 正面面对"断联主义"，回答"连接真的值得吗？" |
| **新要素** | 陆沉登场、静潮社据点可访问、核心辩论 |
| **情感基调** | 沉思、矛盾、带着悲伤的温柔 |

**叙事目的**：
- 呈现"断联主义"的合理性——不是反派，而是另一种选择
- 让玩家思考核心问题：连接真的值得吗？
- 陶夏的成长：从好奇到承担风险
- 林砾的沉默选择：用行动回答

**剧情要点**：

1. **静潮社浮出水面**：
   - 多次遇到路标被拆、信号设备被破坏后，霜期主动找到主角
   - "我可以带你们去见一个人。他能解释为什么会有人做这些。"
   - 霜期带主角前往一个隐蔽的废弃地铁站——陆沉就在那里

2. **陆沉的讲述**：
   - 他不动怒、不激动，只是平静地讲了一个故事——
   - "我管过一个叫'河口'的地方。两百多人。我们和三个聚落做贸易。日子越来越好。然后有一天，一支商队带来了情报，说北边有个聚落出高价收水。我们开了一条新路。两个月后，掠夺者沿着那条路找到了我们。"
   - "河口没了。不是因为掠夺者太强。是因为我们太容易被找到。"
   - "所以你现在明白了吗？每一条路，都是双向的。你连接了希望，也连接了危险。"

3. **陶夏的回应**：
   - 陶夏没有反驳。她想了很久，说：
   - "你说的我都懂。但是……如果河口当时有更多聚落帮忙呢？如果那条路上不只有掠夺者，还有援军呢？"
   - 陆沉："你怎么保证？"
   - 陶夏："……我保证不了。但我可以一直跑下去。"
   - 这是陶夏成长弧的核心台词——从"我想看所有亮灯的地方"到"我愿意为了让灯继续亮而承担风险"

4. **林砾的沉默选择**：
   - 林砾没有参与辩论。但离开时，她把一份手写的路线图留给了陆沉。
   - "这是我们跑过的所有路线。哪些路安全、什么时候有掠夺者活动、哪些聚落需要什么。"
   - "你可以用它来保护你想保护的人。或者你可以用它来决定——哪些路值得重新打开。"
   - 陆沉接过去，没有说话。

5. **静潮社内部分裂**：
   - 霜期告诉主角：静潮社内部有一些激进分子打算破坏信号中继站——这会直接影响鸣砂的广播
   - 主角需要赶去阻止破坏，或者提前修复被破坏的设备
   - **设计意图**：不是"打败敌人"，而是"保护连接"

6. **陆沉的松动**：
   - 章节末尾，陆沉通过霜期传话给主角：
   - "我没有被说服。但我让人把西区隧道的封口拆了。那条路通向一个旧物流站。……算是我欠你的。"
   - 这是陆沉第一次主动做出"连接"的行为

**新增地图节点**：

|---------|------|------|------|------|
| `still_tide_station` | 废弃地铁站（静潮据点） | story | (300, 580) | 陆沉的驻地，Ch.5 后可访问 |

### 第六章：地图的另一半

| 项目 | 内容 |
|------|------|
| **触发条件** | `ch5_done` + 完成 ≥ 45 趟跑商 + ≥ 3 个聚落好感 ≥ Lv2(60) + 收集 ≥ 3 张地图碎片 + `quest_tower_done`（白述线完成） |
| **核心目标** | 拼合天顶地图，确定云海高塔的位置与真实用途 |
| **新要素** | 地图拼合事件、四聚落协作请求、祁岚的信号确认 |
| **情感基调** | 期待、凝聚、即将出发的紧张与激动 |

**叙事目的**：
- 让此前所有跑商积累转化为"意义"
- 四聚落协作——展示玩家一路建立的信任
- 揭示云海高塔的真相
- 为终章做准备

**剧情要点**：

1. **地图拼合**：
   - 白述手中的天顶残图 + 孟回收集的旧票据路线 + 伍拾七发现的地下高速入口图 + 玩家收集的地图碎片
   - 谢令（书院抄录员）协助将所有信息整合到一张完整地图上
   - 当地图拼完的那一刻——所有路线汇聚在同一个点：**云海高塔**

2. **真相揭示**：白述终于完整讲述他所知道的——
   - 云海高塔是灾前的"应急物资分发网络"中枢——一个设计用来在灾难后协调各地物资调配的系统
   - 它还在运转，因为某个人一直在那里维护它
   - "如果重启它的通讯功能，所有聚落之间就不再需要行商来传递信息了。"
   - 陶夏："那我们不就失业了？"
   - 白述微笑："也许。但你们这一路跑出来的信任，机器替代不了。"

3. **四聚落协作**：打通通往高塔的路线需要四个聚落各提供一项关键资源——
   - **北穹塔台**：远程通讯校准模块（让路途中不失联）
   - **温室社区**：长途干粮补给包（路途遥远，需要足够的食物）
   - **废墟游民营地**：地下高速的通行权和向导（伍拾七的人带路）
   - **钟楼书院**：高塔的旧认证密钥（从档案中恢复）
   - 每项资源需要对应聚落好感 ≥ Lv2 才愿意提供
   - **设计意图**：让玩家此前所有的跑商、好感积累、任务完成，在这一刻全部化为"意义"

4. **铁锁的态度**：
   - 姜寒得知主角要去高塔后，罕见地主动来找她们
   - "你们知道那有多远吗？路上没有补给站，没有回头路。"
   - 如果玩家此前对铁锁态度友善（`xiaomao_iron_lock`）：姜寒犹豫之后说"……我可以让小铆跟你们去。她一直念叨着想看看外面。"
   - 如果此前关系紧张：姜寒只是沉默地递过一罐油。"别死在路上。"

5. **祁岚的确认**：
   - 临出发前夜，电台里突然传来一个不是鸣砂的声音——
   - "这里是……云海高塔值守员……编号零七。如果有人能听到这条消息……请不要来。"
   - 陶夏："她说不要去？"
   - 林砾看着记录了几年的频率数字，关掉电台："走吧。"
   - 祁岚的声音是疲惫的、恳求的——她不是在阻止，而是害怕再承受"有人来了又走"的希望落空

6. **出发前夜**：
   - 林砾第一次主动发起篝火对话——
   - "陶夏。如果到了那里，什么都没有呢？"
   - 陶夏："那就当我们多跑了一趟远程。回来继续接单。"
   - 林砾笑了。很轻。
   - "嗯。回来继续接单。"

**解锁**：
- 通往云海高塔的路线在地图上显示（多段，跨越未知区域）
- 终章可触发

**设置 flag**：`ch6_done`、`tower_route_ready`、`all_factions_contributed`

### 终章：云海之上

| 项目 | 内容 |
|------|------|
| **触发条件** | `ch6_done` + 确认出发（玩家主动选择"出发前往云海高塔"） |
| **核心目标** | 完成旅程，回答主题问题，呈现结局 |
| **情感基调** | 安静、辽阔、感动、温暖 |

**路途**：

通往高塔的旅程是一段特殊的长途跑商——没有接单、没有交易，只有路。

- 途经无人区、旧高架、雪原、云层之上的山路
- 沿途穿插两人最后一组对话（自动触发，不需要篝火）
- 每段路触发一段简短回忆——玩家此前做过的关键选择以台词闪回的方式浮现

**对话片段示例**：

```
[雪原路段]
陶夏：你还记得我们第一次到温室的时候吗？
林砾：沈禾说"又来了啊"。
陶夏：对。我当时觉得她不太热情。
林砾：后来呢？
陶夏：后来她每次都给我们留热水。
```

```
[云层入口]
林砾：……我一直没告诉你。
陶夏：嗯？
林砾：山城车库最后一天，有一段无线电。是从西南方向来的。
      它只说了一句话："还有人吗。"
陶夏：……
林砾：我当时想回答。但发射器坏了。
陶夏：所以你一直在追那个信号。
林砾：……嗯。
陶夏把手搭在林砾的肩上。没说话。
```

**到达高塔**：

1. 高塔实际上是一座半嵌入山体的旧通讯设施，顶部天线伸入云层之上
2. 祁岚在门口等着——不是迎接，是拦截
   - "我说了不要来。"她瘦得像纸片人，头发全白了。
   - "你们知道这个地方的意思吗？如果你们重启它，所有聚落都会知道彼此的位置。好人会来。坏人也会来。"
   - 这是最终的主题拷问——和陆沉一样的恐惧，但来自一个守了几十年的人

3. **林砾的回答**（无论玩家选什么，林砾都会说这段话）：
   - "我知道。路是双向的。信号也是。"
   - "但这一路上，我们见过的每一个聚落——他们缺的不只是货物。他们缺的是知道还有别人在。"
   - "所以，就算有风险。我们还是想把灯点亮。"

4. 祁岚没有立刻回答。她转身走进高塔。
   - 门开着。

### 三个结局

结局不取决于终章的单一选择，而是取决于**整个游戏过程中的累积行为**：

#### 结局 A：继续旅行（默认结局）

**条件**：没有任何聚落好感达到 Lv3(90)

**内容**：
- 林砾和陶夏启动了高塔的通讯系统，信号覆盖了她们跑过的所有区域
- 祁岚说这里需要人维护。两人对视一眼。
- 陶夏："我们可以教别人来。但我们不留。"
- 林砾："路上还有单要送。"
- 最后一幕：货车从高塔出发，驶回废土。鸣砂的广播里说："今天的天气——有风。适合出发。"
- **尾声文字**："她们没有停下。也许永远不会。但每一个接到信号的聚落都知道——路的那头，有人还在跑。"

#### 结局 B：定居守望

**条件**：有 ≥ 1 个聚落好感达到 Lv3(90)，但未达成全部 Lv3

**内容**：
- 启动通讯系统后，她们决定把高塔的日常维护交给祁岚和谢令（书院派来的志愿者）
- 两人回到好感最高的那个聚落，把货车停在门口
- 沈禾/韩策/伍拾七/白述（取决于哪个聚落）说："终于肯停下了？"
- 陶夏："也不算停。只是换个方式跑。"
- 她们把货车改装成了那个聚落的"贸易中转站"，帮助培训新的行商
- 最后一幕：货车上画满了不同聚落孩子的涂鸦。林砾坐在车顶记录日志。陶夏在教一个年轻人怎么砍价。
- **尾声文字**："她们停在了一个值得停留的地方。但车还在。总有一天，也许还会再出发。"

#### 结局 C：点亮信号（真结局）

**条件**：4 个主聚落好感均达到 Lv3(90)

**内容**：
- 启动通讯系统后，四个聚落同时收到了信号
- 韩策："信号源定位完成。是真的。"
- 沈禾哭了。
- 伍拾七假装没什么："知道了。"然后偷偷笑了。
- 白述在钟楼敲响了大灾变后的第一声钟。
- 四个聚落各派一人前往高塔，建立永久中继站
- 姜寒带着铁锁商队运来了设备——她说"别误会，只是生意"，但小铆在旁边笑得很开心
- 陆沉没有来，但霜期来了。她在高塔的通讯记录本上写下了第一行字："河口还在。以另一种方式。"
- 最后一幕：鸣砂的广播——第一次不再是"给所有还在路上的人"，而是：
  - "这里是鸣砂。给所有已经到达的人，和所有即将出发的人——"
  - "今天的天气：晴。"
  - 画面拉远。废土上亮起了一盏又一盏灯。
- **尾声文字**："末世没有结束。但信号到达的地方，都有人抬头看向了同一片天空。"

## 十、设计备忘

### 不要做的事

- 不要让任何反派"不可救药"——姜寒有道理，陆沉有苦衷
- 不要让主角被动接受命运——她们始终在主动选择
- 不要让主线催促玩家——"你还在自由跑商？可以的，故事会等你"
- 不要让结局有"坏结局"——三个结局都是积极的，只是方式不同
- 不要让战斗决定剧情走向——决定走向的是贸易、信任和选择

### 一定要做的事

- 让每个聚落的好感积累都在终章产生回响
- 让林砾和陶夏的关系始终是故事的情感核心
- 让"跑商"本身成为叙事行为——每一趟都在重建连接
- 让玩家在终章出发前有"这一路值得"的感觉
- 让鸣砂的最后一句广播成为最温暖的句号

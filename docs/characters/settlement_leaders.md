# 聚落领袖档案

> 四大主聚落的领袖。每人代表一种价值观和生存策略。

---

## 沈禾

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 35~40 岁 |
| 身份 | 温室社区领袖 |
| 聚落 | greenhouse（温室社区） |
| 立绘 ID | `shen_he` |
| 图标 | 🌾 |
| 派系立绘 | `faction_farm` |

### 性格

- 务实朴素，关心具体的事——水、种子、天气、收成
- 对外人冷淡但不恶意，信任需要用实际行动来换
- 重视可持续发展，讨厌浪费

### 说话风格

- 关键词："够不够""能用就行""别浪费"
- 语气平和亲切，不强势但有主见
- 不会直接说"帮我做XX奖励XX"，而是把需求融入自然对话

### 信息权限

- **知道**：温室农业、赵苗的近况、外围农场的事
- **不知道**：塔台的技术细节、废墟的黑市、书院的档案

### 与主角的关系

- 对陶夏的砍价能力有好感（"方法有点……不正常，但有效"）
- 对林砾的务实态度认可
- 温室是主角最早建立信任的聚落之一

### 台词示例

```
"路上辛苦了。这里虽然不大，但至少能吃上热饭。"
"水泵的事……不急，但也不能一直拖着。你下次路过塔台的时候，帮我问一声吧。"
"行商就是行商，合理。"
```

## 韩策

| 项目 | 设定 |
|------|------|
| 性别 | 男 |
| 年龄 | 40~45 岁 |
| 身份 | 北穹塔台领袖 |
| 聚落 | tower（北穹塔台） |
| 立绘 ID | `han_ce` |
| 图标 | 🔧 |
| 派系立绘 | `faction_tech` |

### 性格

- 理性、数据导向，对不专业的人没耐心
- 尊重能力，不尊重身份——你行就是行，不行数据说话
- 严谨到近乎苛刻，但并非冷酷——他关心的是效率和结果
- 可能有技术或军事背景

### 说话风格

- 关键词："数据""效率""不够稳定""你们的运力……"
- 说话条理清晰，逻辑严密
- 会用数据怼人："你们的运力不如铁锁稳定"
- 不夸人，但沉默就是认可

### 信息权限

- **知道**：塔台设备数据、程远的报告、信号频率
- **不知道**：温室种植计划、废墟地下通道

### 与主角的关系

- 初期冷脸——"两个小姑娘能有多少运力？"
- 被实际交付能力打动后逐渐尊重
- 与林砾在技术话题上有一定共鸣

### 台词示例

```
"你们的运力不够稳定。铁锁商队每月初五准时到，你们呢？"
"我的设备接收到过。坐标指向西南方。但那可能只是回声，不值得冒险。"
```

## 伍拾七

| 项目 | 设定 |
|------|------|
| 性别 | 男 |
| 年龄 | 30~35 岁 |
| 身份 | 废墟营地头目 |
| 聚落 | ruins_camp（废墟营地） |
| 立绘 ID | `wu_shiqi` |
| 图标 | 🔩 |
| 派系立绘 | `faction_scav` |

### 性格

- 江湖气，粗犷豪爽，直来直去
- 假装不在乎但其实讲义气——嘴硬心软
- 对规则和体制不信任，偏爱自由和实力说话
- 对手下有保护欲——尤其对刀鱼，嘴上说"这丫头烦死了"，实际是事实上的监护人

### 说话风格

- 关键词："我的地盘""信你一次""规矩"
- 说话直接，偶尔粗鲁但真诚
- 不玩虚的，承诺了就会兑现

### 信息权限

- **知道**：废墟的地形、阿锈的消息、黑市行情
- **不知道**：塔台的技术、温室的储备量、书院的藏书

### 与主角的关系

- 跟陶夏的江湖气投缘，偶尔互相贫嘴
- 对林砾保持警惕尊重——"这姑娘不简单"
- 铁锁是他的眼中钉——拒绝统一定价

### 台词示例

```
"有人在传那是旧文明的什么中枢。也有人说那是陷阱。你猜我信哪个？"
"我的地盘我说了算。铁锁想在这儿定价？做梦。"
```

## 白述

| 项目 | 设定 |
|------|------|
| 性别 | 男 |
| 年龄 | 50~55 岁 |
| 身份 | 钟楼书院领袖 |
| 聚落 | bell_tower（钟楼书院） |
| 立绘 ID | `bai_shu` |
| 图标 | 📖 |
| 派系立绘 | `faction_scholar` |

### 性格

- 博学睿智，重视知识传承和文明保存
- 说话慢条斯理，爱引经据典
- 温和但固执——在知识保存这件事上不让步
- 唯一会认真回应"远方信号"话题的领袖

### 说话风格

- 关键词："档案里记载""旧时代""记住了就不会忘"
- 文雅深沉，有学者风范
- 不急不慢，愿意花时间解释事情

### 信息权限

- **知道**：书院档案、苏墨的发现、旧文明知识
- **不知道**：聚落商业机密、铁锁行动计划

### 与主角的关系

- 认为铁锁只关心利润，不关心文明——态度冷淡
- 对主角愿意跑商运送文物/书籍的行为心存感激
- 与谢令是师父与学生（也是事实上的父女关系）

### 台词示例

```
"如果那个信号是真的……它可能是灾前'应急物资分发网络'的核心节点。我的档案里有相关记载。"
```

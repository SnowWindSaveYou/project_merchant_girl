## 前哨站管理者

### 赵苗

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 25~30 岁 |
| 身份 | 外围农场管事 |
| 聚落 | greenhouse_farm（外围农场） |
| 立绘 ID | `zhao_miao` |
| 图标 | 🌱 |
| 上级 | 沈禾 |

### 性格与说话风格

- 勤劳朴实，乐观，手上永远有土
- 沈禾的学生，对师父崇拜和信赖
- 说话热情直接，容易不好意思
- 会热情地招待行商——"太好了！下次来我请你们吃烤玉米！"

### 剧情钩子

- 在外围农场挖到了一条干净的地下水脉

### 台词风格

```
"抱歉抱歉……我是赵苗，这片外围农场归我管。沈禾让我在这儿开荒的。"
"嘿嘿，现在已经不会犯那种错了！大概……"
"是啊！师父是最好的人！"
```

## 聚落常驻 NPC

### 季微（纪微）

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 22~25 岁 |
| 身份 | 北穹塔台信号技术员 |
| 聚落 | tower（北穹塔台） |
| 立绘 ID | `ji_wei` |
| 图标 | 📡 |

### 性格与说话风格

- 话少但善意，比韩策更愿意主动帮助外人
- 韩策的技术助手，信号监测站值班员
- 塔台的"暖面孔"——帮主角获取韩策不愿直接给的技术支持
- 偶尔透露韩策不愿公开的信息

### 剧情钩子

- 她最早发现"云海高塔"方向的规律信号，但被韩策要求保密

### 性格与说话风格

### 刀鱼

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 13~15 岁 |
| 身份 | 少女拾荒者 |
| 聚落 | ruins_camp（废墟营地） |
| 立绘 ID | `dao_yu` |
| 图标 | 🔪 |

### 性格

- 表面油嘴滑舌、爱吹牛，内心极度渴望被认可
- 总想证明自己"不比大人差"，但被夸时反而手足无措
- 不记得父母，从记事起就在废墟翻东西
- 被伍拾七捡回营地前独自活了至少两年——伍拾七是她事实上的监护人，但她绝不承认

### 说话风格

- 机灵、爱吹牛、带着得意劲
- 嘴上叫伍拾七"老大"，心里当家长

### 与主角的关系

- 对陶夏有强烈的"姐姐崇拜"——觉得陶夏会说话会砍价还到处跑
- 对林砾是安静的信任——林砾答应"下次给你讲讲"城市的事，她记得很牢

### 剧情钩子

- 偷偷收藏了一本画册，里面画的是灾前城市的样子——但她从没见过真正的城市
- 她的摊位偶尔出现"刀鱼精选"——她认为很值钱的旧时代物件（可能是垃圾也可能真值钱）

### 剧情价值

> "在废墟里长大的孩子，依然可以对世界充满好奇。"
> 她不悲惨——她很快乐，只是那种快乐建立在不知道自己失去了什么的基础上。

### 谢令

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 18~20 岁 |
| 身份 | 钟楼书院抄录员 |
| 聚落 | bell_tower（钟楼书院） |
| 立绘 ID | `xie_ling` |
| 图标 | 🖊 |

### 性格

- 安静内向，做事极其认真到近乎执拗
- 不擅长和陌生人说话，但话题涉及感兴趣的内容（外面的世界、旧文明）时会突然话很多，说完又会因为"说太多了"而脸红
- 自幼被白述收养在书院长大，没离开过钟楼方圆三公里
- 她对世界的全部认知都来自档案里的文字——既博学又天真

### 与主角的关系

- 对陶夏充满好奇和羡慕——陶夏是"活的百科全书"
- 年龄相近，陶夏的外向能带动谢令说话

### 与白述的关系

- 师徒也是父女
- 白述从不限制她，但谢令自己选择留在书院——"这些字如果我不抄，就没人抄了"

### 剧情钩子

- 整理档案时发现一封未寄出的信——寄给云海高塔某个值班员的私人信件

### 剧情价值

> "通过文字认识世界的人，也会渴望亲眼去看。"
> 陶夏用脚丈量世界，谢令用笔记住世界。

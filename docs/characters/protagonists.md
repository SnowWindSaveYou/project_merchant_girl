## 陶夏

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 约 20 岁 |
| 身份 | 行商搭档（谈判/社交/采购） |
| 立绘 ID | `taoxia` |
| 表情差分 | normal / happy / sad / surprised / angry / thinking |

### 性格

- 活泼外向，好奇心旺盛，情绪表达丰富
- 擅长社交和砍价，是队伍的"嘴"
- 不是傻白甜——面对真正的危险时会收起笑容，认真时语气会变沉
- 善于观察对方的情绪变化，经常捕捉到林砾藏不住的小表情

### 说话风格

- 多用问句和感叹句，语气生动
- 喜欢用反问句吐槽："又不给钱的事？""你每次都这样回答。"
- 偶尔正经起来的时候很有分量，比如"我想听你的理由"
- 陶夏在认真的时候，林砾反而会安静下来听

### 背景

- 出发前在温室社区附近生活
- 是她主动拉林砾出来跑商的（"是你拉我出来的"——林砾语）
- 对旧世界有模糊的好奇但没有执念——她更在意眼前的人

### 与林砾的互动模式

| 状态 | 模式 |
|------|------|
| 日常 | 陶夏主动发话 → 林砾简短回应 → 陶夏追问或吐槽 → 林砾偶尔一句意外的话 |
| 严肃 | 角色反转——林砾主动开口意味着她认真了，陶夏会安静听 |
| 分歧 | 不吵架，而是沉默——做饭时不说话、睡觉时背对着 |

### 对其他角色的态度

- **沈禾**：尊重，觉得她务实可靠
- **韩策**：被怼过但不记仇，反而觉得有趣
- **伍拾七**：江湖气投缘，偶尔互相贫嘴
- **白述**：有点畏惧文化人，但聊起来很投入
- **刀鱼**：姐姐对妹妹的喜爱，觉得她很可爱
- **谢令**：年龄相近，陶夏见多识广，谢令博学多才，互补
- **霜期**：同龄人之间最有思辨张力的对话——信任 vs 隔离

### 写台词时注意

```
✅ 好的台词：
"哇！好久没吃到新鲜蔬果了！"
"你每次都这样回答。"
"我想听你的理由。"
"你说到炮塔的时候是不是眼睛亮了一下？"

❌ 避免的台词：
过度卖萌的无脑乐观
"好棒哦~""太厉害了吧~" 这种偶像剧风格
面对危险还嘻嘻哈哈
```

## 林砾

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 约 22 岁 |
| 身份 | 行商搭档（驾驶/机修/安全） |
| 立绘 ID | `linli` |
| 表情差分 | normal / happy / sad / surprised / angry / thinking |

有着漂亮的修长手指。

### 性格

- 理性冷静，话少而精，实用主义
- 行动派，不喜欢空谈
- 情绪内敛——感情通过行动和细微反应表达，而非语言
- **反差萌核心**：平时正经严肃，但在特定话题上会暴露出女孩子的一面
  - 聊到炮塔/机械时眼睛会亮
  - 修好东西后会偷偷高兴
  - 把事情排得整整齐齐时格外满足
  - 被陶夏抓到这些瞬间时会口是心非地否认
  - 爱干净

### 说话风格

- **省略号开头是标志性特征**："……嗯。""……不好说。""……没有。"
- 句子简短，多用陈述句，少用感叹号
- 技术话题时话量明显增加，甚至会嘟嘟囔囔自言自语
- 冷幽默——不是讲笑话，而是一本正经地说出很好笑的话
  - "……你敢卖试试。"
  - "上次刹车的时候，你的创意收纳砸到我脚了。"
- 极少主动表达感情；当她主动开口说较长的话时，说明她认真了

### 背景

- 出发前在"山城车库"做机修学徒
- 离开时老师傅送了一本维修手册，最后几页是空的，等她自己填
- 从山城出来那天起一直在追踪一个远方信号（Ch.2 揭示）
- 末世生存经验丰富——以前独自搜刮过资源点，没人望风

### 反差萌 / 暴露瞬间模板

教程气泡和篝火对话中已经建立的范式：

| 场景 | 林砾的流露 | 陶夏的捕捉 | 林砾的否认 |
|------|-----------|-----------|-----------|
| 货车改装 | 说到炮塔时语速加快 | "你说到炮塔的时候是不是眼睛亮了一下？" | "……没有。" |
| 收音机 | 修好后偷偷笑了 | "等下你在笑什么？" | "……没。就是修了很久，能收到信号还挺高兴的。" |
| 自动计划 | 排计划时格外专注开心 | "你排计划的时候感觉特别开心是我的错觉吗？" | "……把事情安排得整整齐齐的，有什么不好。" |
| 搜刮 | 提到以前独自搜刮 | "现在有我了！你搜，我望风！" | "……嗯。那确实方便多了。" |

**规律**：
1. 林砾的流露是不自觉的（语速变化 / 表情变化 / 无意识的话）
2. 陶夏敏锐地捕捉到并直接指出
3. 林砾用省略号 + 简短否认/转移来回应
4. 关键是否认得不够彻底——"……没有。" 而不是真的生气

### 对其他角色的态度

- **陶夏**：保护欲强，默默关心但不会说出口
- **沈禾**：尊重，觉得务实的人好打交道
- **韩策**：技术上互相尊重，但不喜欢他的傲慢
- **伍拾七**：警惕但不敌意
- **白述**：愿意听他讲旧文明的事
- **小铆**：技术上的惺惺相惜——两人聊引擎和零件时眼睛都会亮

### 写台词时注意

```
✅ 好的台词：
"……先加油。出发前油箱不满的话，我会睡不着。"
"……你敢卖试试。"
"……不知道。先记下来。"
"……够的。一直够的。"

❌ 避免的台词：
长篇大论的解释说明（说明书式台词）
主动表达感情（"我很开心""我担心你"）
夸张的语气词（"哇！""天啊！"）
失去冷静地大喊大叫（她的愤怒是冷的）
```

## 麻薯号（货车）

| 项目 | 设定 |
|------|------|
| 全名 | 麻薯号 |
| 命名者 | 陶夏 |
| 类型 | 改装大货车，两人的移动据点和家 |

### 命名由来

陶夏坚持要给货车取名字，理由是"天天一起走，没有名字多没礼貌"。林砾觉得给车起名很幼稚，但拗不过。陶夏的取名逻辑：外面硬邦邦、里面软乎乎，跟她们的车一样——外壳全是焊疤和铁板，但车厢里被收拾得很舒服。

林砾的态度变化：嫌俗 → 懒得反驳 → 习惯了 → 不自觉地也开始叫"麻薯"（被陶夏抓到时会否认）。

### 叙事功能

- **日常化的情感锚点**：对货车的称呼体现两人关系进度——早期林砾只说"车"，中期开始说"那个车"，后期不小心说出"麻薯"
- **危机时的情绪放大器**：货车受损 = 家被毁，修车 = 重建家园，比抽象的"据点受损"更有代入感
- **角色表达工具**：陶夏会跟车说话（"麻薯今天辛苦了！"），林砾会在修车时自言自语技术细节——两种截然不同的"对车说话"方式

### 写台词时注意

```
✅ 好的台词：
陶夏："麻薯今天发动的时候声音不太对，是不是感冒了？"
林砾："……发动机不会感冒。是化油器进沙了。"
陶夏（修好后）："麻薯你又活过来了！"
林砾（独自修车时）："……左边这个螺栓得换了。再跑两趟就该断了。"

❌ 避免的台词：
两人同时用拟人化方式称呼货车（这是陶夏专属）
林砾当面用"麻薯"（只在不经意间流露）
```

## 音乐设定

### 分工

| 角色 | 音乐能力 | 说明 |
|------|---------|------|
| 陶夏 | 唱歌 | 什么都唱，废土自编歌、旧世界残曲、即兴哼唱；偶尔跑调但情绪到位 |
| 林砾 | 口琴 | 在废墟里找到的旧口琴，10 个簧片坏了 2 个，她自己磨的替代片，音色有点涩 |

### 口琴的来历

林砾在某次独自搜刮资源点时找到的，塞在一个生锈的铁盒里。10 个簧片坏了 2 个，她用废金属片自己磨了替代品——音不算准，但能用。她从没提过这件事，是陶夏某天翻车厢时发现的。

### 场景设计

| 场景 | 描写方式 | 叙事效果 |
|------|---------|---------|
| **日常行车** | 陶夏随口哼歌，林砾偶尔加一段口琴伴奏 | 旅途的松弛感，让"又一段赶路"不沉闷 |
| **反差萌** | 陶夏发现林砾一个人在偷偷吹口琴 | "等下你刚才在吹口琴？" "……没有。" "我都听到了！" |
| **沉默时刻** | 两人吵完架/遇到危险后，林砾默默吹一段 | 不用说话，音乐替代了语言——她在说"我在这" |
| **高光时刻** | 某个篝火夜，陶夏唱歌 + 林砾伴奏，完整合作一首 | 关系里程碑，两人第一次真正的合奏 |

### 关于口琴的细节

- 2 个磨出来的簧片让某些音有辨识度很高的"涩感"，成为林砾的标志音色
- 口琴本身破旧但被擦得很干净——和林砾对待所有工具的态度一样
- 这个设定与麻薯号呼应：不完美但能用，修修补补但一直在路上

### 写台词时注意

```
✅ 好的台词：
陶夏："你那个口琴是不是有两个音吹不出来？"
林砾："……能吹。只是音准差一点。我自己磨的簧片。"
陶夏："哇，你还会磨簧片？！"
林砾："……不难。就是个金属加工。"

❌ 避免的台词：
林砾主动提议合奏（应该是被动的、自然发生的）
陶夏嘲笑林砾的口琴技术（她会觉得很酷，不会嘲笑）
```

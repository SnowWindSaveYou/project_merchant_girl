# 势力 NPC 档案

> 主线剧情中的新势力角色。Ch.3 起陆续登场。

---

## 铁锁商队

> **理念**: "自由贸易是混乱之源，只有统一管控的贸易才能避免流血。"
>
> **组织形态**: 3~5 辆重型改装货车编队行动，有护卫、调度、定价权。
>
> **设计意图**: 代表"善意的控制"——不是劫匪，确实在维持废土贸易秩序，只是方式是垄断。

### 姜寒（铁锁商队队长）

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 38~42 岁 |
| 身份 | 铁锁商队队长 |
| 登场 | Ch.3 |
| 状态 | 尚无立绘 |

#### 外观

- 束发，旧军大衣改的商队外套
- 皮带上挂计算器和短波对讲机
- 左手无名指缺失一节
- 气质严厉而疲惫
- *反向约束: 避免女将军感、魅惑风、华丽饰品、少女化*

#### 性格

- 冷静果断，信奉"规则比善意可靠"
- 对手下有母亲般的保护欲
- 年轻时也是自由行商，搭档在一次贸易纷争中被杀——从此认为只有垄断定价才能防止暴力

#### 对主角态度变化

| 阶段 | 态度 |
|------|------|
| 初期 | 鄙视："两个小姑娘能活多久？" |
| 中期 | 尊重："你们居然还没死" |
| 后期 | 欣赏但固执 |

#### 转变关键

当她看到主角不靠垄断也能让聚落和平交易时，信念开始松动。但真正让她改变的，是小铆说"我不想变成一个被人害怕的商人"。

### 小铆（铁锁商队年轻机械师）

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 16~18 岁 |
| 身份 | 铁锁商队机械师 |
| 登场 | Ch.3 |
| 状态 | 尚无立绘 |

#### 外观

- 油渍工装裤，绑在额头的旧焊接护目镜
- 总在口袋里掏零件，笑起来露虎牙
- 手上有机油污渍但不脏，是"正在工作中"的状态
- *反向约束: 避免性感化、偶像系、军装化*

#### 性格

- 好奇心旺盛，技术天赋极高
- 内心向往自由但害怕背叛姜寒
- 代表"下一代不必重复上一代的恐惧"

#### 与林砾的关系

技术上的惺惺相惜——两人聊起引擎和零件时眼睛都会亮。林砾是小铆第一个在铁锁之外交到的朋友。

#### 功能

铁锁内部的"裂缝"，为主角提供铁锁内部信息；后期可能脱离铁锁加入贸易联盟。

## 静潮社

> **理念**: "旧世界因为过度连接而崩溃，每个聚落应该自给自足，不再互相依赖。"
>
> **组织形态**: 没有统一指挥，各地有独立的"守静人"，通过留言和暗号松散联络。
>
> **设计意图**: 代表"恐惧驱动的善意"——不是在害人，而是在用错误的方式保护人。

### 陆沉（守静人代表）

| 项目 | 设定 |
|------|------|
| 性别 | 男 |
| 年龄 | 48~55 岁 |
| 身份 | 静潮社守静人代表 |
| 登场 | Ch.5 |
| 状态 | 尚无立绘 |

#### 外观

- 灰白短发、旧棉服、围巾遮半张脸
- 手持旧步话机（关着的）——像某种纪念物
- 坐姿或靠墙站立，不像在警戒，更像在等什么人
- *反向约束: 避免反派化、武装感、宗教领袖感*

#### 性格

- 温和但坚定，说话慢条斯理，从不提高声音，但几乎不让步
- 对主角不敌视，甚至有些心疼："你们还这么年轻，不要重蹈覆辙"
- 他反对的不是两个女孩，而是"连接"这个行为本身

#### 背景

曾是某个繁荣贸易聚落的管理者。一次大规模交易引来掠夺者，聚落被毁，家人在混乱中失散。从此成为"断联主义"的践行者。

#### 转变关键

当主角帮助一个聚落抵御危机——不是靠武力，而是靠贸易网络的互助——他开始动摇。

### 霜期（静潮社脱离者）

| 项目 | 设定 |
|------|------|
| 性别 | 女 |
| 年龄 | 19~22 岁 |
| 身份 | 静潮社脱离者 |
| 登场 | Ch.4（短暂接触），Ch.5（正式） |
| 状态 | 尚无立绘 |

#### 外观

- 灰绿斗篷，布缠短刀，写满笔记的手臂绷带
- 短发利落，一侧别着旧金属发夹
- 眼神警觉但藏着好奇
- *反向约束: 避免杀手/刺客化、暴露装、可爱化*

#### 性格

- 在静潮社长大，内心认同"保护"的出发点，但逐渐怀疑"断联"是否有效
- 代表"在恐惧中长大的人也可以选择信任"

#### 与陶夏的关系

年龄相近但成长环境截然相反——陶夏在流动中长大相信连接，霜期在封闭中长大被教导隔离。两人的对话是主线中最有思辨张力的段落。

## 势力与聚落关系

### 铁锁商队 × 聚落

| 聚落 | 关系 |
|------|------|
| 北穹塔台 | 主要客户，韩策默许铁锁垄断部分技术品流通 |
| 温室社区 | 被动依赖，沈禾不喜欢铁锁高价但冬季只有铁锁送货 |
| 废墟营地 | 对抗，伍拾七拒绝接受统一定价 |
| 钟楼书院 | 冷淡，白述认为铁锁只关心利润 |

### 静潮社行为模式

- 拆除路标
- 堵塞废弃隧道入口
- 散布"贸易带来掠夺者"的言论
- 偶尔破坏信号中继设备

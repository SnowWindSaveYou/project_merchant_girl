# 流浪NPC档案

> 不固定在任何聚落，在废土上游荡的独行者。  
> 他们是主角旅途中的"路上的人"——缘分到了就遇见，走远了就消失。

---

## 孟回

### 基础信息

| 项目 | 设定 |
|------|------|
| **身份** | 行脚医生 |
| **性别** | 女 |
| **定位** | 流动NPC / 前期出现 / 温暖系角色 |
| **系统ID** | `meng_hui` |
| **立绘** | `image/portrait_meng_hui_20260406000106.png` |
| **Q版** | `image/chibi_npc_meng_hui_20260409120916.png` |
| **图标** | 💊 |
| **UI配色** | 文字 `rgb(128, 168, 128)` / 背景 `rgba(32, 44, 36, 0.94)` |
| **偏好出现位置** | 聚落节点、中转节点 |
| **迁移概率** | 每趟行程后 50% 概率迁移 |

### 外观关键词

披着拼接雨披、背药箱、穿旧风衣、推单轮货架、温柔关切的表情

### 性格核心

**"停不下来的老毛病"** —— 孟回是一个被愧疚和善良同时驱动的人。

- 灾前是医院实习医生，导师让她先撤离，自己留守，再也没出来
- 这份"被留下的人"的愧疚变成了行动力：路上遇到受伤的人就治，治完就走
- 她选择不在任何聚落定居，因为"看到路上还有人在受苦，就坐不住了"
- 不觉得自己是好人，只是"停不下来"
- 像主角的一种未来投影：同样在流动，但已经失去了"想停下来的地方"

### 说话方式

| 特征 | 说明 |
|------|------|
| **语气** | 温柔但克制，不煽情 |
| **习惯** | 先关心对方身体状况，再说别的 |
| **专业感** | 谈医疗时语气利落、判断精准，有实习医生的底子 |
| **情感表达** | 通过行动而非语言（塞药、帮忙修车、不声不响地帮忙） |
| **脆弱时刻** | 提到导师时会失控，眼眶红一瞬然后迅速别过脸 |

**✅ 符合人设的台词**：
- "别动。伤口我看一下。"（开口就是诊断）
- "怕。但路上受伤的人更怕。"（承认恐惧，但不退缩）
- "什么好人。就是个停不下来的老毛病。"（拒绝被标签化）
- "也好。很久没跟人一起吃饭了。"（犹豫后接受温暖）
- "举手之劳，下次路上见。"（帮完就走，不纠缠）

**❌ 不符合人设的台词**：
- "我一定会拯救所有人的！"（太中二，她知道自己救不了所有人）
- "你们应该感谢我。"（她不求回报）
- "没什么大不了的。"（她不会轻描淡写别人的痛苦）

| 深层 | NPC_MH_004 | 无法治愈的病 | 请林砾帮忙→导师的故事→药方笔记被水泡坏→需要去书院找文献补全 |

### 剧情价值

| 阶段 | 作用 |
|------|------|
| **前期** | 教学期的温暖角色，提供路线情报和一次性高利润订单 |
| **中期** | 带来跨聚落消息碎片（模糊的各地传闻、路况、市价） |
| **后期** | Ch.6 地图碎片来源之一（`孟回的最后一张地图`事件） |
| **终局** | 孟回收集的旧票据路线是抵达天顶的四块拼图之一 |

### 信息权限

| 知道什么 | 不知道什么 |
|---------|-----------|
| 各地传闻（模糊）、路况、市价 | 聚落内部事务、领袖的私下态度 |

### 与主角的关系

- 陶夏叫她"孟回姐"，关心她独自露营安不安全
- 林砾对她的医学知识有专业层面的尊重（会主动问药材来源）
- 孟回是唯一一个让林砾也偶尔主动开口关心的NPC

### 主题意义

**"如果我不路过呢？"** —— 孟回代表了一种选择：明知道自己救不了所有人，仍然选择在路上。她是对"行商"这一行为的另一层诠释——主角运货物，孟回运医术，本质上都是"在路上传递必要之物"。

## 鸣砂

### 基础信息

| 项目 | 设定 |
|------|------|
| **身份** | 独行商人 / 无线电播报员 |
| **性别** | 男 |
| **定位** | 流动NPC / 中期出现 / 世界观串联者 |
| **系统ID** | `ming_sha` |
| **立绘** | `image/portrait_ming_sha_20260406000227.png` |
| **Q版** | `image/chibi_npc_ming_sha_20260409121030.png` |
| **图标** | 🎒 |
| **UI配色** | 文字 `rgb(178, 148, 98)` / 背景 `rgba(44, 38, 28, 0.94)` |
| **偏好出现位置** | 聚落节点、资源节点 |
| **迁移概率** | 每趟行程后 50% 概率迁移 |
| **电台频道** | `mingsha`（三频道之一） |

### 外观关键词

背大包、戴帽子、穿旧户外夹克、狡黠而友善的笑容。前期只闻其声不见其人（通过电台），后期现身时穿旧播报制服、披短斗篷、背便携天线包。

### 性格核心

**"生意归生意，但你们是例外"** —— 鸣砂是一个用嬉皮笑脸包裹伤疤的人。

- 来自沙漠另一端的"盐湖城"贸易站，穿越三百公里无人区来到这片区域
- 曾经有过同伴，被背叛后定了三条规矩：不合伙、不欠人情、永远留退路
- 表面油滑世故，实际上最看重信义——只是不再轻易付出信任
- 经营电台广播不是为了赚钱，而是"沙漠里的沙丘风吹过会响，像大地在说话，我希望电台也是这样"
- 嘴上说"一锤子买卖"，实际上每次都会给主角多留一点好处

### 说话方式

| 特征 | 说明 |
|------|------|
| **语气** | 嬉皮笑脸、商人的圆滑、偶尔冒出一句哲理 |
| **口头禅** | "生意归生意"、"信不信由你"、"反正我已经赚过一波了" |
| **幽默方式** | 自嘲型——"不跟比我穷的人做生意。开玩笑的。" |
| **认真时刻** | 收起嬉皮笑脸，直视对方的眼睛，语速变慢 |
| **广播语气** | 克制、信息密度高、偶尔带一丝人情味但从不煽情 |

**✅ 符合人设的台词**（面对面）：
- "嗬，同行？难得在这条路上碰到行商。"（老江湖打招呼）
- "消息当然不是白给的。不过看在同行的面子上——这次算我请。"（口硬心软）
- "有过同伴。后来发现——人心比路上的野兽更难预测。"（伤疤）
- "你小丫头嘴真贫！行，你们是例外。"（被陶夏逗笑后松口）
- "你是我见过的第一个想这么大的行商。"（收起笑脸，认真评价）

**✅ 符合人设的台词**（电台广播）：
- "这里是鸣砂，正在为旅途中的各位播报。"（固定开场）
- "祝每一位在路上的人，平安到达目的地。"（固定结语）
- "收到一位匿名听众来信：'谢谢鸣砂，路上听到你的声音就觉得没那么孤单了。'……嗯，谢谢你。我也是。"（广播里唯一流露的温柔）
- "今天的音乐时间。可惜我只有一张唱片，所以……又是这首。"（自嘲式温情）

**❌ 不符合人设的台词**：
- "我是最厉害的商人！"（他不需要证明自己）
- "别碰我的东西！"（太小气，他的世故是大方中带防线）
- "我早就看穿一切了。"（他不是愤世嫉俗，只是不再轻信）

### 对话模式（4段式深入）

| 阶段 | 对话ID | 标题 | 内容要点 |
|------|--------|------|---------|

### 电台广播系统

鸣砂经营三频道之一的`mingsha`频道，共12条广播内容：

| 类型 | 数量 | 示例 |
|------|------|------|
| 天气/路况 | 3条 | 风向、落石报告、行车提醒 |
| 市场情报（带奖励） | 3条 | 温室净水充裕、塔台零件涨价、新电池 |
| 旧时代故事 | 2条 | "手机"的故事、名字由来 |
| 温情/个人 | 3条 | 匿名听众来信、唯一的唱片、祝平安 |
| 生存知识 | 1条 | 车辆过热处理 |

**广播语气规范**（来自叙事指南）：
- 克制、信息密度高
- 偶尔带一丝人情味但从不煽情
- 固定开场"这里是鸣砂"
- 随主线推进从"冷旁观"变成"主动提醒主角"

| **终局A** | 最后一幕广播："今天的天气——有风。适合出发。" |

### 与主角的关系

- 称陶夏"小丫头"，被她的嘴贫逗笑后认定"你们是例外"
- 林砾对他保持警惕（"你也是行商？"），但逐渐承认他的情报价值
- 鸣砂是唯一一个让主角意识到"废土之外还有更大世界"的角色

### 主题意义

**"路永远在那儿，等着有人走。"** —— 鸣砂代表了"连接"本身。他的电台把孤立的聚落变成了一个网络，他的商路把已知世界和未知世界连接起来。如果说主角是"运送货物的人"，鸣砂就是"运送声音的人"——最终都是在修复断裂的文明纽带。

## 祁岚（后期关键NPC）

> ⚠️ 祁岚严格意义上不是"流浪NPC"，但她同样不属于任何聚落，且在主线后期才出现，故收录于此。

### 基础信息

| 项目 | 设定 |
|------|------|
| **身份** | 云海高塔看守者 |
| **定位** | 后期主线关键角色 / 终局真相揭示者 |
| **首次登场** | 不直接见面，先以信号、传闻、旧笔记被多次提及 |

### 外观关键词

苍白、瘦削、旧防寒披肩、携带老式升降机钥匙与记录匣

### 性格核心

**疲惫、孤独、对"重建"既期待又怀疑。**

- 掌握高塔真正用途与旧文明最后的运输计划
- 决定是否让主角继续靠近，取决于玩家此前如何处理各聚落关系
- 让终局不是单纯"抵达高塔"，而是"理解为什么还要继续连接别人"

### 说话方式

低声、缓慢、像在说故事，也像在试探听者。

### 剧情价值

| 阶段 | 作用 |
|------|------|
| **Ch.2起** | 通过信号、传闻、旧笔记被反复提及（"西南方向有塔还在发光"） |
| **Ch.6** | 临出发前夜，电台里传来一个不是鸣砂的声音——祁岚的首次直接出现 |
| **终局** | 负责终局真相揭示、结局分支选择、世界观收束 |

### 设计意义

终局BOSS不是怪物，而是一个问题：**"你为什么还要继续连接别人？"** 祁岚是提出这个问题的人。

## 流浪NPC出场时间线

```
前期（教学期）
  ├── 孟回出现 → 温暖、治疗、路线情报
  └── 鸣砂（仅电台声音）→ 建立世界联系感

中期（扩展期）
  ├── 孟回 → 跨聚落消息碎片
  ├── 鸣砂现身 → 市场情报、世界观扩展
  └── 鸣砂广播提及"云海高塔"

后期（汇聚期）
  ├── 孟回 → "最后一张地图"事件（终局拼图之一）
  ├── 鸣砂 → 特别播报（主线广播链高潮）
  └── 祁岚 → 信号→传闻→直接出现→终局
```

## 流浪NPC vs 聚落NPC 对比

| 维度 | 聚落NPC | 流浪NPC |
|------|---------|---------|
| **位置** | 固定在某个聚落 | 每趟行程后可能迁移 |
| **遇见方式** | 到达聚落必定出现 | 缘分——到了才知道在不在 |
| **情报范围** | 深但窄（只了解本聚落） | 广但浅（各地传闻、路况、市价） |
| **叙事功能** | 展示聚落内部生态 | 串联不同聚落、扩展世界边界 |
| **与主角关系** | 基于好感度的固定成长 | 更随性——每次相遇都是一段小故事 |

# 流浪NPC档案

> 不固定在任何聚落，在废土上游荡的独行者。  
> 他们是主角旅途中的"路上的人"——缘分到了就遇见，走远了就消失。

### 外观关键词

披着拼接雨披、背药箱、穿旧风衣、推单轮货架、温柔关切的表情

### 性格核心

**"停不下来的老毛病"** —— 孟回是一个被愧疚和善良同时驱动的人。

- 灾前是医院实习医生，导师让她先撤离，自己留守，再也没出来
- 这份"被留下的人"的愧疚变成了行动力：路上遇到受伤的人就治，治完就走
- 她选择不在任何聚落定居，因为"看到路上还有人在受苦，就坐不住了"
- 不觉得自己是好人，只是"停不下来"
- 像主角的一种未来投影：同样在流动，但已经失去了"想停下来的地方"

### 说话方式

| 特征 | 说明 |
|------|------|
| **语气** | 温柔但克制，不煽情 |
| **习惯** | 先关心对方身体状况，再说别的 |
| **专业感** | 谈医疗时语气利落、判断精准，有实习医生的底子 |
| **情感表达** | 通过行动而非语言（塞药、帮忙修车、不声不响地帮忙） |
| **脆弱时刻** | 提到导师时会失控，眼眶红一瞬然后迅速别过脸 |

**✅ 符合人设的台词**：
- "别动。伤口我看一下。"（开口就是诊断）
- "怕。但路上受伤的人更怕。"（承认恐惧，但不退缩）
- "什么好人。就是个停不下来的老毛病。"（拒绝被标签化）
- "也好。很久没跟人一起吃饭了。"（犹豫后接受温暖）
- "举手之劳，下次路上见。"（帮完就走，不纠缠）

**❌ 不符合人设的台词**：
- "我一定会拯救所有人的！"（太中二，她知道自己救不了所有人）
- "你们应该感谢我。"（她不求回报）
- "没什么大不了的。"（她不会轻描淡写别人的痛苦）

### 外观关键词

背大包、戴帽子、穿旧户外夹克、狡黠而友善的笑容。前期只闻其声不见其人（通过电台），后期现身时穿旧播报制服、披短斗篷、背便携天线包。

### 性格核心

**"生意归生意，但你们是例外"** —— 鸣砂是一个用嬉皮笑脸包裹伤疤的人。

- 来自沙漠另一端的"盐湖城"贸易站，穿越三百公里无人区来到这片区域
- 曾经有过同伴，被背叛后定了三条规矩：不合伙、不欠人情、永远留退路
- 表面油滑世故，实际上最看重信义——只是不再轻易付出信任
- 经营电台广播不是为了赚钱，而是"沙漠里的沙丘风吹过会响，像大地在说话，我希望电台也是这样"
- 嘴上说"一锤子买卖"，实际上每次都会给主角多留一点好处

### 说话方式

| 特征 | 说明 |
|------|------|
| **语气** | 嬉皮笑脸、商人的圆滑、偶尔冒出一句哲理 |
| **口头禅** | "生意归生意"、"信不信由你"、"反正我已经赚过一波了" |
| **幽默方式** | 自嘲型——"不跟比我穷的人做生意。开玩笑的。" |
| **认真时刻** | 收起嬉皮笑脸，直视对方的眼睛，语速变慢 |
| **广播语气** | 克制、信息密度高、偶尔带一丝人情味但从不煽情 |

**✅ 符合人设的台词**（面对面）：
- "嗬，同行？难得在这条路上碰到行商。"（老江湖打招呼）
- "消息当然不是白给的。不过看在同行的面子上——这次算我请。"（口硬心软）
- "有过同伴。后来发现——人心比路上的野兽更难预测。"（伤疤）
- "你小丫头嘴真贫！行，你们是例外。"（被陶夏逗笑后松口）
- "你是我见过的第一个想这么大的行商。"（收起笑脸，认真评价）

**✅ 符合人设的台词**（电台广播）：
- "这里是鸣砂，正在为旅途中的各位播报。"（固定开场）
- "祝每一位在路上的人，平安到达目的地。"（固定结语）
- "收到一位匿名听众来信：'谢谢鸣砂，路上听到你的声音就觉得没那么孤单了。'……嗯，谢谢你。我也是。"（广播里唯一流露的温柔）
- "今天的音乐时间。可惜我只有一张唱片，所以……又是这首。"（自嘲式温情）

**❌ 不符合人设的台词**：
- "我是最厉害的商人！"（他不需要证明自己）
- "别碰我的东西！"（太小气，他的世故是大方中带防线）
- "我早就看穿一切了。"（他不是愤世嫉俗，只是不再轻信）

## 雪冬

### 基础信息

| 项目 | 设定 |
|------|------|
| **身份** | 自封邮递员 / 废土唯一的信使 |
| **性别** | 女 |
| **定位** | 流动NPC / 中前期出现 / 信件系统核心 |
| **系统ID** | `xue_dong` |
| **立绘** | `image/portrait_xue_dong_20260412064247.png` |
| **Q版** | `image/chibi_npc_xue_dong_20260412064256.png` |
| **图标** | 📮 |
| **UI配色** | 文字 `rgb(168, 188, 208)` / 背景 `rgba(32, 38, 48, 0.94)` |
| **偏好出现位置** | 各聚落节点、中转节点 |
| **迁移概率** | 每趟行程后 50% 概率迁移 |
| **老家** | 北穹塔台（韩策的聚落） |

### 外观关键词

背着大邮包、戴旧邮差帽、穿拼接风衣、鞋底磨得很薄、眼神躲闪但不是因为害怕而是害羞

### 性格核心

**"说不出口的话，就跑着送过去"** —— 雪冬是一个用行动代替语言的人。

- 在塔台长大，帮韩策整理过旧时代的信件档案，对"信"这个概念有特殊执念
- 面对面交流时容易紧张、结巴、说到一半就低头——不是社恐，是找不到合适的词
- 但写信时完全不同：文字温暖、细腻、观察敏锐，像换了一个人
- 选择当邮递员有一个理由说不出口：她曾经有一封信没来得及送出去，收信人就再也见不到了
- 她相信"人跟人之间不该断了联系"，所以只要有信要送，不管多远都跑着去

### 说话方式

| 特征 | 说明 |
|------|------|
| **语气** | 紧张、断续、常用省略号，句子短 |
| **习惯** | 先低头翻包，翻到信才抬头说话；说完正事就想走 |
| **反差** | 写信/便签时文字流畅、温暖、有画面感，完全是另一种表达力 |
| **情感表达** | 通过行动（跑很远去送信）而非语言；偶尔在便签里流露真实想法 |
| **脆弱时刻** | 提到"没寄出的信"时会沉默很久，然后快速换话题 |

**✅ 符合人设的台词**（面对面）：
- "呃……你、你们是林砾和陶夏吧？有信……给你们。"（紧张递信）
- "不用谢。我就是……跑一趟而已。"（低估自己的付出）
- "字？我的字不好看。沈禾的字才好看。"（不自信但会偷偷在意）
- "嗯……那，那我走了。路上……小心。"（告别时也是断续的）
- "……三天。从温室走过来的。路上还好。"（被问到路程时轻描淡写）

**✅ 符合人设的文字**（便签/书信）：
- "路上看到一棵开花的树。不知道是什么花。但我觉得它一定等了很久才开的。"
- "韩策的字很难认，但墨迹压得很重。写得用力的人，一般都是认真的。"
- "这封信沈禾写了两遍。第一遍被水泼了。她说没关系再写一遍。但我觉得第一遍写得更好。"
- "有时候走在路上会想，如果有一天所有人都不用写信了，是不是说明大家离得够近了。那其实挺好的。"

**❌ 不符合人设的台词**：
- "我是邮递员，有什么信尽管交给我！"（太自信外向）
- "这封信很重要，你一定要好好看！"（她不会评价别人的信件内容）
- "我一个人走在路上不害怕的。"（她会害怕，只是信比害怕更重要）

- 初期紧张，面对陶夏的热情有些招架不住
- 林砾的沉默反而让她放松——"不用说很多话也没关系"的同类
- 陶夏逐渐成为她最自在的对话对象（"你说话好多……但我不讨厌"）
- 林砾会默默帮她检查邮包的搭扣有没有松（机械师的本能）

### 主题意义

**"如果这封信能送到，是不是就没那么远了？"** —— 雪冬代表了"连接"的最原始形式：不靠电波、不靠技术，只靠一个人跑着把一封信从这里带到那里。如果说鸣砂用声音连接世界，雪冬就是用脚步连接世界。她的存在本身就是对"废土"的一种反驳——只要还有人愿意送信，人与人之间就没有真正断开。

## 祁岚（后期关键NPC）

> ⚠️ 祁岚严格意义上不是"流浪NPC"，但她同样不属于任何聚落，且在主线后期才出现，故收录于此。

### 基础信息

| 项目 | 设定 |
|------|------|
| **身份** | 云海高塔看守者 |
| **定位** | 后期主线关键角色 / 终局真相揭示者 |
| **首次登场** | 不直接见面，先以信号、传闻、旧笔记被多次提及 |

### 外观关键词

苍白、瘦削、旧防寒披肩、携带老式升降机钥匙与记录匣

### 性格核心

**疲惫、孤独、对"重建"既期待又怀疑。**

- 掌握高塔真正用途与旧文明最后的运输计划
- 决定是否让主角继续靠近，取决于玩家此前如何处理各聚落关系
- 让终局不是单纯"抵达高塔"，而是"理解为什么还要继续连接别人"

### 说话方式

低声、缓慢、像在说故事，也像在试探听者。

### 剧情价值

| 阶段 | 作用 |
|------|------|
| **Ch.2起** | 通过信号、传闻、旧笔记被反复提及（"西南方向有塔还在发光"） |
| **Ch.6** | 临出发前夜，电台里传来一个不是鸣砂的声音——祁岚的首次直接出现 |
| **终局** | 负责终局真相揭示、结局分支选择、世界观收束 |

### 设计意义

终局BOSS不是怪物，而是一个问题：**"你为什么还要继续连接别人？"** 祁岚是提出这个问题的人。

## 流浪NPC出场时间线

```
前期（教学期）
  ├── 孟回出现 → 温暖、治疗、路线情报
  ├── 雪冬出现 → 引入信件系统、第一封NPC来信
  └── 鸣砂（仅电台声音）→ 建立世界联系感

中期（扩展期）
  ├── 孟回 → 跨聚落消息碎片
  ├── 雪冬 → 投递悬念信件、送信委托开放
  ├── 鸣砂现身 → 市场情报、世界观扩展
  └── 鸣砂广播提及"云海高塔"

后期（汇聚期）
  ├── 孟回 → "最后一张地图"事件（终局拼图之一）
  ├── 雪冬 → 个人故事线揭示（没寄出的信）
  ├── 鸣砂 → 特别播报（主线广播链高潮）
  ├── 雪冬 → 终局：写给鸣砂的广播稿
  └── 祁岚 → 信号→传闻→直接出现→终局
```

## 流浪NPC vs 聚落NPC 对比

| 维度 | 聚落NPC | 流浪NPC |
|------|---------|---------|
| **位置** | 固定在某个聚落 | 每趟行程后可能迁移 |
| **遇见方式** | 到达聚落必定出现 | 缘分——到了才知道在不在 |
| **情报范围** | 深但窄（只了解本聚落） | 广但浅（各地传闻、路况、市价） |
| **叙事功能** | 展示聚落内部生态 | 串联不同聚落、扩展世界边界 |
| **与主角关系** | 基于好感度的固定成长 | 更随性——每次相遇都是一段小故事 |

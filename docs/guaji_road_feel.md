# 《末世行商》公路感设计文档
### 让旅途本身成为叙事 —— 路景、沉默、小事与呼吸

---

## 零、问题诊断

当前旅途体验：**进度条 + 每 30 秒一句对话 + 电台 = 等待到达**。

公路文的灵魂不在于"路上发生了什么大事"，而在于：

> **路本身有表情。**
> 天光在变、废墟在后退、引擎在响、两个人偶尔说一句话、偶尔什么都不说。
> 到达是暂停，不是目的。

当前系统缺失的不是"更多事件"，而是 **感官层**——让玩家觉得自己真的坐在一辆小货车里，穿过一片有温度的废土。

---

## 一、设计理念

### 三层旅途体验

```
第三层：事件（现有）   随机事件 / 伏击 / 发现       ← 偶发，0-3 次/趟
第二层：人声（现有）   chatter 对话 / 电台广播      ← 间歇，每 30 秒
第一层：路景（缺失）   风景描写 / 天气 / 车内氛围    ← 持续，填满空白
```

**第一层"路景"是地基**。没有它，对话和事件都漂在空中。

### 核心原则

1. **不加系统，加内容** —— 路景不是新机制，是现有 chatter 系统的一个新频道
2. **不需要交互** —— 路景文本自动出现、自动消失，玩家不需要点任何东西
3. **不抢对话** —— 路景在对话间隙出现，对话优先级永远更高
4. **有就更好，没也不亏** —— 和 chatter/radio 一样的设计哲学

## 二、路景系统（Scenery）

### 2.1 什么是路景

一行简短的环境描写文本，出现在车窗/进度条附近，停留 5-6 秒后淡出。没有说话人，没有气泡框——它是"车窗外的世界"。

**视觉呈现**：
- 字体比 chatter 小一号，颜色偏灰（如 `{160,160,150,180}`）
- 无边框、无背景板，像一行旁白
- 位置：进度条下方或 chatter 气泡上方（不占对话空间）
- 可选：前面加一个小图标（ 🌤 / 🌧 / 🌙 / 🏚 ）区分类别

### 2.2 触发规则

```
时机：chatter 对话消失后 8 秒内，如果没有新 chatter 排队，插入一条路景
频率：每趟最多 6 条路景（比 chatter 更多，因为更轻）
间距：两条路景之间至少 20 秒
优先级：chatter > 路景（chatter 出现时立即收起路景）
```

## 三、沉默时刻（Silence Moments）

### 3.1 什么是沉默时刻

公路文里最动人的段落，往往是什么也没发生的时候。

**沉默时刻**是一种特殊的"空事件"：没有文字、没有对话、没有路景——只有进度条在走、引擎在响。但在沉默之前或之后，会出现一条特殊提示，让沉默变得有意义。

### 3.2 实现方式

```
触发条件：连续 45 秒没有 chatter、路景、事件
概率：40%（不是每次都有，避免刻意）
表现：
  1. 先出现一行浅灰色小字，停留 4 秒：
     "两个人都没说话。路在车轮下面，安静地走。"
  2. 然后 20 秒的空白——什么都不出现
  3. 空白结束后，恢复正常的 chatter/路景 轮转
```

## 四、路途小事件（Roadside Moments）

### 4.1 和随机事件的区别

| | 随机事件（现有） | 路途小事件（新增） |
|---|---|---|
| **后果** | 影响资源/战斗/flag | **纯叙事，无实质影响** |
| **交互** | 需要选择/操作 | 最多一个可选按钮 |
| **情绪** | 紧张/兴奋 | **安静/温暖/好奇** |
| **作用** | 丰富游戏性 | **丰富旅途质感** |

## 十、设计备忘

### 文字风格指南

- **短**：每条路景控制在 15-35 字，一眼扫完。不是散文，是速写
- **具体**：写"防护栏锈成了棕红色"，不写"路边有些破旧的东西"
- **克制**：不抒情、不感叹、不解释。陈述事实，让玩家自己感受
- **有呼吸**：句子之间留白。用句号，不用省略号（省略号是 chatter 的语气，路景是旁白的语气）
- **不重复 chatter 的视角**：chatter 是"两个人在说话"，路景是"窗外的世界"。不要在路景里写对话

### 参考调性

> 路面上有旧车辙。比我们的轮胎宽得多。不知道是谁留下的。

> 引擎声稳定而低沉，像某种安静的陪伴。

> 两个人都没说话。但这种沉默不难受。

如果每条路景读完让人想深呼吸一下——那就对了。

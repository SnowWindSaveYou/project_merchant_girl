# 待开发功能清单

> 记录已规划但尚未实现的系统功能，供后续迭代参考。

---

## 1. 天气系统

**优先级**: 中  
**关联模块**: `events/event_scheduler.lua`, `core/flow.lua`, `ui/screen_travel.lua`

### 现状
- 天气预报情报已可兑换（`intel.lua` weather 类型），但只产出装饰性文本
- 没有实际天气状态机，旅途中不存在天气变化

### 设计方向
- **天气状态机**: 晴朗 / 沙尘 / 雨雾 / 霜冻，按行程随机切换
- **天气对玩法的影响**:
  - 沙尘：移动速度 -20%，燃料消耗 +15%，视野降低
  - 雨雾：捷径危险度 +1 级，事件触发率 +10%
  - 霜冻：货物损耗（冷藏模块可抵消），轮胎抓地力下降
  - 晴朗：无额外效果（基准）
- **天气预报情报增强**: 提前显示下趟旅途的天气，让玩家决定出发时机或更换车辆配件
- **UI 表现**: 旅途界面顶部天气条，NanoVG 粒子效果（沙尘/雨滴）

## 2. 路况系统

**优先级**: 中  
**关联模块**: `map/world_graph.lua`, `events/event_scheduler.lua`

### 现状
- 边类型（main_road / path / shortcut）已存在，但只影响显示样式和事件触发概率
- 没有动态路况变化

### 设计方向
- **路况状态**: 良好 / 泥泞 / 积雪 / 塌方 / 封锁
  - 泥泞：速度 -30%，轮胎磨损
  - 积雪：速度 -40%，燃料 +20%，需防滑链
  - 塌方：临时封路，需绕行（可清理，消耗时间+资源）
  - 封锁：掠夺者封路，需战斗/贿赂/绕行
- **动态变化**: 路况按游戏内"周期"变化，每 N 趟行程刷新
- **地图显示**: 已知路况在地图上用图标/颜色标注

## 3. 车辆适配系统（天气/路况换装）

**优先级**: 中-低  
**关联模块**: `truck/modules.lua`, `ui/screen_garage.lua`

### 现状
- 卡车模块升级已完成（引擎/货仓/雷达/冷藏/炮塔 5 个模块）
- 没有针对天气/路况的专项适配配件

### 设计方向
- **新增配件槽**: 轮胎 / 底盘附件
  - 轮胎类型：普通轮胎 / 防滑链 / 越野轮胎 / 沙地轮胎
  - 底盘附件：挡泥板（减少泥泞惩罚）/ 防风罩（减少沙尘影响）
- **配件获取**: 聚落商店购买 / 事件奖励 / 废墟营地拆解
- **换装时机**: 在聚落车库中更换，旅途中不可换
- **维护/耐久**: 配件有耐久度，使用后需维修或更换

## 4. 车辆维护与磨损

**优先级**: 低  
**关联模块**: `truck/modules.lua`

### 设计方向
- 模块在使用后逐渐磨损，性能随耐久度下降
- 需要在聚落花费金钱/零件维修
- 磨损到 0 后模块失效（不是损毁，可修复）
- 增加"行商的日常成本感"，与收入形成循环

## 5. 前哨站功能扩展

**优先级**: 低  
**关联模块**: `settlement/`

### 现状
- 4 个前哨站（温室农场、穹顶哨站、地铁营地、旧教堂）已有定价和基础交易
- 缺少独特功能

### 设计方向
- 每个前哨站有 1-2 个独特服务（如：温室农场可种植，地铁营地可拆解装备）
- 前哨站也可有好感度系统（较主聚落简化）

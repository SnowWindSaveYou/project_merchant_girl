# 末世行商 — 美术风格指南（图标篇）

## 整体风格定位

**手绘手账风 · 彩色铅笔 · 贴纸感**

灵感来源：手账本 / Bullet Journal / 旅行日记中的彩色贴纸插画。
所有 UI 图标必须保持统一的视觉语言，不可出现纯黑白素描或写实插画。

## 风格对照表

### ✅ 正确风格

- 外轮廓清晰但不生硬（手绘感的粗细变化）
- 内部有彩色铅笔填色（暖棕、暖黄、淡红、橄榄绿等）
- 整体偏暖色调
- 白色描边形成"贴纸"感
- 造型简洁可爱、略带 Q 版比例
- 单个图标，居中放置

### ❌ 错误风格

- 纯黑色墨水线条，无彩色填充
- 多个图标散布在画面中
- 写实比例（如写实信封、写实帐篷）
- 灰色/冷色调为主
- 过于复杂的细节
- 无白色描边

## NPC Chibi 头像风格

NPC 头像遵循独立的 chibi 风格规范（非本文档范围），但需注意：
- chibi 头像在按钮中以圆形裁剪显示（`iconRound = true`）
- 尺寸通常为 24px（按钮内）或 28px（独立显示）
